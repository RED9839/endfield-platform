# 아크라이트 (Arclight) — 전기 / 한손검 / 뱅가드 ★5

주스탯 **민첩**, 보조 **지능**. 전기 뱅가드 — 감전 소모 + **초단쿨(3초) 연계 게이지 수급** + 팀 전기 증폭(지능 비례). 궁에 게이지 회복 없는 유일 뱅가드 → 연계 의존. 1.2 장방이(전기 스트라이커) 파티 고정.

---
## 스킬 (설명문 원문)

### 기본 공격 「추적 사냥」
최대 5단 물리. 강일 16 불균형. 처형 400→900%.

### 배틀 스킬 「질풍 섬광」 (게이지 100)
> 적 옆 순간이동 2회 베기 소량 물리. **감전 상태면 감전 소모 + 추가 1회 전기 + 게이지 회복**.
- 1/2단계 45→101% / 추가 180→405% / 게이지 회복 30→40 / 불균형 5+5

### 연계 스킬 「천둥의 울림」 (쿨 **3초**, 조건: 적 감전 또는 감전 소모됨)
> 순간이동 연속 베기 물리 + 게이지 회복. **초단쿨 = 감전만 유지하면 3초마다 8~10 게이지 + 궁 에너지.**
- 155→350% / 게이지 회복 8→10 / 궁 에너지 +5→10 / 불균형 5 / 쿨 3초

### 궁극기 「천둥번개」 (게이지 90)
> 전기 아크 돌진, 경로 전기 + **전기 부착**. 일정 시간 후 아크 폭파 전기. 전기 부착 적이면 **전기 부착 소모 → 감전 부여**.
- 1단계 156→350% / 2단계 244→550% / 불균형 7+7

## 재능 (설명문 원문)
- **황무지의 방랑자**: 질풍 섬광으로 추가 효과 **3회 발동 시**, 지능 1당 팀 전기 피해 +0.05%/**+0.08%**(15초). (잠재5: 발동 2회로 완화 / 잠재3: ×1.3)
- **만물의 지혜**: 아츠 부착 부여 시 30%/**50%** 확률 면역.

## 🎮 현재 구현 (DD류 — `app/game/dd/roster.ts` arclight, Lv1 계수)
- OP_BASE: cls **vanguard**, hp 2689, attack 110(튜닝), speed **71**(민첩), ultCost 90.
- **질풍 섬광**(배틀, power 0.9, `shockBonus:{power:1.8, gauge:30}`) — 감전 적이면 추가 전기 + 게이지 30. 검증 ✅
- **천둥의 울림**(연계, power 1.55, `requires shock`, **cooldown 1**(3초≈1턴), `gaugeGain:8`) — 핵심 초단쿨 수급. 검증: 감전 전 불가 → 후 가능 ✅
- **천둥번개**(궁, power 4.0, `element:electric`, `attach:electric`, `forceShock`, selfUlt) — 전기 부착 + 강제 감전. 검증: 궁 후 shock ✅
- 엔진 신규: `shock` 상태(감전, applyAttach 감전 시 자동 부여) · `shockBonus`(감전 소모 추가타) · `forceShock`(강제 감전).
- **황무지의 방랑자**(정식 구현): 질풍 감전 소모 **3회마다**(`procCount`) → 팀 전기 증폭 = **`gearGrade × 0.0008`**(15초≈3턴, 비중첩). **지능 → 장비 등급(gearGrade) 치환**. 검증: 장비등급 60 → **+4.8%**(=지능 61 ×0.08% 동일 magnitude) ✅
- **만물의 지혜**(정식 구현): **적이 아크라이트에게 아츠 부착 시 50% 확률로 부착 무효**(`artsImmune:0.5`, `applyAttach`에서 `Math.random()` 롤). 방어형. ⚠ **현재 DD 적은 아츠 부착 미모델 → 휴면**(메커니즘만 검증, 적 아츠 부착 추가 시 자동 작동). 검증: 200회 중 ~88 면역(≈50%) ✅
- ⚠ 단순화: 질풍의 감전 소모는 shock 유지(천둥 3초 spam 위해), 기본공격 처형, 잠재(명함 제외).

## 🔑 장비 등급 치환 (gearGrade)
4스탯(힘/민첩/지능/의지)을 뺀 대신 **`gearGrade`(장비 등급 통합값, 명함 ~60)** 로 스탯 비례 재능을 치환. "지능 1당 +0.08%" → "장비등급 1당 +0.08%". 향후 장비 시스템에서 gearGrade를 장착 등급 합으로 산출하면 자동 연동(여풍 돈오·로시·자이히 등 모든 스탯 비례 재능 공통).
