# 아크라이트 (Arclight) — 전기 / 뱅가드 ★?

※ 위키 원문 배율표는 미보유(구현 기반 복원). 감전 소모 순간이동 타격 + 무료 연계 게이지 수급 + 팀 전기 증폭. (전기 뱅가드)

## 스킬
- **배틀 질풍 섬광** (단일): 순간이동 2회 베기 소량 물리. **감전 적이면 감전 소모 + 추가 전기 + 게이지 회복**.
- **연계 천둥의 울림** (cond: 감전/감전 소모 후): 순간이동 연속 베기 물리 + 게이지 회복. (사실상 무료=에너지 환급, 궁충)
- **궁극 천둥번개** (돌진 경로 AoE): 전기 아크 돌진 경로 전기 + 전기 부착. 일정 시간 후 아크 폭파 추가 전기. (딜궁, 버프 아님)

## 재능
- **황무지의 방랑자**: 질풍 섬광 후 지능 비례 팀 전기 피해↑(= 장비 등급 비례 팀 오라).
- **만물의 지혜**: 아츠 50% 면역.

## 🎮 현재 구현
- passive: `{ teamStatAmp: 0.07, damageResist: 0.10 }`
  - `teamStatAmp 0.07` = 황무지의 방랑자(지능=장비 등급 비례 팀 피해 오라) / `damageResist 0.10` = 만물의 지혜
- 연계 = `LINK_ENERGY_REFUND { arclight: 1 }`(코1-환급1=무료) + `LINK_ULT_GAIN_ARCLIGHT 5`(예외: 일반 연계 궁충 10, 아크라이트만 5)
- 궁극 = 데미지(ULT_KIND 특수 아님). ULT_ENERGY 90
