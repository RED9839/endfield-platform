# 안탈 (Antal) — 전기 / 아츠 유닛 / 서포터 ★4

주스탯 **지능**, 보조 **힘**. 전기/열기 취약(장기) + 전기/열기 증폭궁. 열기·전기팟 범용. ("탈 4성")

## 스킬
- **기본 교환 전류**: 4단 전기. 강일 15 불균형.
- **배틀 지정 연구 대상** (게이지 100): 적 60초 포커싱 89%→200% 전기 + **전기 취약 + 열기 취약**(5~10%, **60초**). 단일 1명.
- **연계 자기 폭풍 실험장** (쿨 25~24초): **포커싱 적이 물리이상/아츠부착 상태일 때**. 151%→340% 전기 + 그 물리이상/아츠부착 **재부여**(아츠 폭발 유도). 불균형 10.
- **궁극 오버클럭 타임** (게이지 100): 팀 전체 **전기 증폭 + 열기 증폭**(8~20%, 12초).

## 재능
- **즉흥적인 천재성**: 증폭 상태 팀원이 스킬 피해 시 **[72~108 + 힘×0.6~0.9]** 회복(30초 1회).
- **무의식**: 30% 확률 물리 면역 + 자기 회복 [27~45 + 힘×].

## 🎮 현재 구현
- skillMechanic `combo-strike`(연계 전기 부착 자동)
- passive: `{ healOnCast: 10, damageResist: 0.12, targetVuln: 0.10 }`
  - `targetVuln 0.10` = 지정 연구 대상 전기/열기 취약(빠져 있던 핵심). ⚠ 0.12는 sim 47→44% → 0.10이 47% + 위키 취약 10% 일치
  - `healOnCast 10` = 즉흥적인 천재성 / `damageResist 0.12` = 무의식
- ULT_KIND **buff**(전기/열기 증폭), ULT_ENERGY 100
