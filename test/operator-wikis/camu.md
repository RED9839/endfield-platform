# 카뮤 (Camu) — 열기 / 장병기 / 뱅가드 ★6 (한정)

주스탯 **민첩**, 보조 **지능**. 열기 부착·취약·증폭·게이지·연타·허약 다 부여. 추적 메커니즘.

## 스킬
- **기본 피의 속죄**: 5단 열기. 강일 18 불균형.
- **배틀 사르는 불꽃** (게이지 100): 핏빛 날개 89%→200% 열기 소범위 + **열기 부착 + 허약 + 열기 취약**(5~7%). 처치 시 다른 적으로 이동(부착+허약+취약). 배회 적 연계 명중 시 폭발(45~100%). 불균형 10.
- **연계 영혼의 가시** (쿨 20~18초): **열기 부착 소모/흡수 후**. 133%→300% 열기 + 게이지 16~20. 추적 222%→500% + 추가 불균형/게이지.
- **궁극 선혈의 비** (게이지 130): 267% 열기 광역 + 열기 부착 + 게이지 32. **사용 후 다음 배틀 스킬 = 추적(연계 취급, 게이지 미소모)**, 추적 상태 15초. 불균형 15.

## 재능
- **죄를 쫓는 자**: 배회 적 연계 명중 시 메인 **[60+지능×0.3]** 회복 + 연타(15초). 추적 시 배회 무관 발동.
- **혈류 소생**: 스킬로 자기 회복 시마다 열기 피해 +4%(최대 5스택), 팀 25% 공유. 만피면 2스택.

## 🎮 현재 구현
- passive: `{ teamAmp: 0.10, grantMultiHit: true, healOnCast: 6, targetVuln: 0.08 }`
  - `teamAmp 0.10` = 혈류 소생(팀 열기 피해 오라)
  - `grantMultiHit` = 연타 / `healOnCast 6` = 죄를 쫓는 자(회복, passivePlayEffect가 장비 등급 비례 처리 → [60+지능×0.3]의 지능 비례 대응)
  - `targetVuln 0.08` = 사르는 불꽃 열기 취약(설명만 있고 미적용이던 것 추가)
- ULT_ENERGY 130. 추적 = makeChaseCard + transformed(3턴), cardFromDeck가 camu 배틀→chase
