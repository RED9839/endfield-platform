# 카뮤 (Camille) — 열기 / 장병기 / 뱅가드 ★6 (한정)

주스탯 **민첩**, 보조 **지능**. **첫 6성 한정 뱅가드 "다 떡칠"** — 열기 부착·열기 취약·허약·게이지 수급·연타·회복·핏빛 날개(배회 디버프). 레바테인 파티 필수급. 살카즈(뱀파이어).

## 스킬 (설명문 원문)

### 기본 공격 「피의 속죄」
최대 5단 열기. 강일 18 불균형. 처형 400→900%.

### 배틀 스킬 「사르는 불꽃」 (게이지 100)
> 핏빛 날개 소환, 소범위 열기 + **열기 부착**. 날개가 목표 주변 **배회**하며 **허약 + 열기 취약** 부여. 목표 처치 시 다른 적으로 이동. 배회 적이 카뮤 연계 명중 시 폭발 추가 열기.
- 89→200% / 허약 5→7% / 열기취약 5→7% / 폭발 45→100% / 불균형 10 / 날개 45초

### 연계 스킬 「영혼의 가시」 (쿨 20~18초, 조건: 적 **열기 부착 소모/흡수 후**)
> 가로지르며 열기 + 게이지 회복.
- 133→300% / 게이지 16→20 / 추적 222→500% / 추적 게이지 32→40 / 쿨 20→18초

### 궁극기 「선혈의 비」 (게이지 130)
> 창의 비 + 가로베기 열기 + **열기 부착** + 게이지. **사용 후 다음 배틀 = 추적**(연계 취급, 게이지 무소모, 15초).
- 267%(고정, 무성장) / 게이지 32 / 불균형 15

## 재능 (설명문 원문)
- **죄를 쫓는 자**: 영혼의 가시가 핏빛 날개 배회 적 명중 시 **[60+지능×0.3]** 회복 + **연타**(15초). 추적은 배회 무관 발동.
- **혈류 소생**: 스킬로 자기 회복 시마다 열기 피해 +2%/**+4%**(최대 5스택), 팀 25%. 만피면 2스택씩.

## 잠재능력 (명함 제외)
01 겁화의 세례(허약/취약 +5%·날개 +15초) / 03 파쇄하는 가시(연계 쿨-2초·×1.3·게이지×1.15) / 04 수호자의 증명(궁 에너지-15%) / 05 피로 맺은 결속(혈류 소생 스택당 +6%)

## 🎮 현재 구현 (DD류 — `app/game/dd/roster.ts` camu, Lv1 계수)
- OP_BASE: cls **vanguard**, hp 2689, attack 110(튜닝), speed **80**(민첩), ultCost **130**.
- **사르는 불꽃**(배틀, power 0.89, `element:heat`, `attach:heat`) — `apply`: **허약 0.05**(`applyBuff weaken`) + **열기취약 0.05**(`bumpVuln heat`) + **핏빛 날개**(`wing` 상태). 검증: 부착1·허약0.95·열취0.05·날개 ✅
- **영혼의 가시**(연계, power 1.33, `requires anomalyConsumed`(열기부착 소모), cooldown 4, `gaugeGain:16`) — **죄를 쫓는 자**: 날개 적 명중 시 회복([60+gearGrade×0.3]) + 연타. 검증: 1000→1078·연타+1 ✅
- **선혈의 비**(궁, power 2.67, `element:heat`, `attach:heat`, target row, gaugeGain 32, ultCost 130).
- 엔진 신규: **`healUnit`(첫 힐 시스템)** + **혈류 소생**(자기 회복 시 열기 +4% 최대 5스택, 팀 25%) · `wing` 상태(핏빛 날개) · `죄를 쫓는 자`(연계+날개 → 회복/연타).
- ⚠ 단순화: 추적 교체(궁 후 배틀→추적)·핏빛 날개 폭발·처치 시 날개 이동 미모델, 기본공격 처형, 잠재.
