# 아델리아 (Ardelia) — 자연 / 아츠 유닛 / 서포터 ★6 (배포)

주스탯 **지능**, 보조 **의지**. 부식 부여→소모로 물리·아츠 취약 + 회복. 아츠 부착 없음(부식 직접 부여).

## 스킬
- **기본 바위의 속삭임**: 4단 자연. 강일 18 불균형.
- **배틀 질주하는 돌리** (게이지 100): 142%→320% 자연. **부식 적 → 부식 소모 + 물리 취약 + 아츠 취약**(12~20%, **30초**). 불균형 10.
- **연계 화산 분화** (쿨 18~17초): **방불/아츠부착 없는 적에 강일 후**. 45%→100% 자연 + 화산 구름 추적 폭발 111%→250% + **부식 부여**(7초).
- **궁극 복슬복슬 파티** (게이지 90): 73%→165% 자연 다단 광역(돌리 분신). 불균형 2~3.

## 재능
- **친구의 그림자**: 배틀/궁 돌리 그림자 생성 → 메인 닿으면 **[45~90 + 의지×0.38~0.75]** 회복(만피면 최저 HP 아군). 최대 10개.
- **마운틴 서퍼**: 배틀 추가 효과 발동 시 근처 부식 적 있으면 배틀 1회 더(주동 1회).

## 🎮 현재 구현
- skillMechanic `corrosion-support`(스킬=자연 데미지 카드, 회복은 재능)
- passive: `{ healOnCast: 16, corrodeConsumeVuln: 0.18, noArtsAttach: true }`
  - 부식 사이클(useRunState ardelia 블록): 연계 = 부식 강제 부여(corrodeVuln 0.12) / 배틀 = 부식 소모 → `corrodeConsumeVuln 0.18`(물리+아츠 취약)
  - `healOnCast 16` = 친구의 그림자(장비 등급 비례) / `noArtsAttach` = 자연 단일 부착 안 함(부식 직접 부여형)
- ULT_ENERGY 90. 마운틴 서퍼(배틀 재발동)은 미모델
