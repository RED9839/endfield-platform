# 아델리아 (Ardelia / 전작 에이야퍄들라의 리컨비너) — 자연 / 아츠 유닛 / 서포터 ★6 (배포)

**첫 서포터 · 유일 6성 배포 자연 서포터**(만능). 부식 셋업→소모로 **물리+아츠 취약 동시(30초 장지속)** + 돌리 그림자 회복. **아츠 부착이 없어** 부착 운영 파티(레바테인·라스트 라이트)와 무충돌. 주스탯 **지능**, 보조 **의지**. 카프리니(로도스). 힐/딜보조 양립·높은 범용성.

---
## 스킬 (설명문 원문)

### 기본 공격 「바위의 속삭임」
최대 4단 자연. 강력한 일격 18 불균형. **처형 400→900%**.

### 배틀 스킬 「질주하는 돌리」 (게이지 100)
> 돌리 타고 돌진 자연. 목표가 **부식** 상태일 때 부식 소모 + 30초간 **물리 취약 + 아츠 취약** 부여.
- 142→320% / 취약 12→20% / 30초 / 불균형 10

### 연계 스킬 「화산 분화」 (쿨 18~17초)
> 메인이 **방어 불능·아츠 부착 없는 적**에 강력한 일격 후. 화산 구름 던져 자연 + 추적 폭발 → 주변 적에 절반 자연 + **강제 7초 부식**.
- 본체 45→100% / 폭발 111→250% / 불균형 10 / 쿨 18→17초

### 궁극기 「복슬복슬 파티」 (게이지 90, 쿨 15초)
> 3초 지속 시전, 돌리 분신 무작위 사방 투척 다단 자연(0.3초당 1회). 재능으로 확률 돌리 그림자(회복).
- 73→165% / 불균형 2(Lv10+ 3) / 지속 3초

## 재능 (설명문 원문)
- **친구의 그림자**(0정~): 배틀 명중 시 돌리 그림자 3개 / 궁 착지 시 10% 확률. 메인이 닿으면 **[45+의지×0.38](0정)→[90+의지×0.75](2정)** 회복. 만피면 최저 HP% 아군. 최대 10개·10초.
- **마운틴 서퍼**(2정): 배틀 부식 소모 발동 시 근처 다른 부식 적 있으면 **즉시 배틀 1회 더**(주동 사용당 1회).
- 장비 조합 I/II/III(파랑/보라/노랑).

## 잠재능력 (명함 제외)
01 양들의 천국(부식 소모 취약 +8%) / 02 게임 보상(친구의 그림자 최저 HP 1명 추가 치유 50%) / 03 격렬한 분출(궁 +1초·그림자 확률 ×1.2) / 04 바위틈의 꽃(궁 에너지 -15%) / 05 화산 증기(연계 쿨 -2초·피해 ×1.2·부식 +4초)

## 🎮 현재 구현 (DD류 — `app/game/dd/roster.ts` ardelia, Lv1 계수)
- OP_BASE: cls **supporter**(첫 서포터), hp 2689, attack 110, speed 62, ultCost **90**. 주스탯 지능.
- **질주하는 돌리**(배틀, power 1.42, `element:nature`) — `apply`: 부식 적(`corrosion` 상태) → 소모 + **물리/아츠 취약 0.12**(30초≈6턴). id훅: **친구의 그림자** 최저 체력% 아군 회복(`90+gearGrade×0.75`=135). 검증: 회복+135 ✅
- **화산 분화**(연계, power 1.56, `target:all`, cooldown 4, `requires` 방불 0·아츠 부착 0) — 자연 + `apply`: **강제 부식**(`corrosion` + vuln.all 0.12). 검증: 게이트 true·부식상태 부여 ✅
- **복슬복슬 파티**(궁, power 2.19=73%×3, `target:all`, ultCost 90) — 광역 다단 자연 + 친구의 그림자 회복(궁은 확률 근사 ×0.5).
- 엔진: **`corrosion` 상태 마커 정비** — applyAttach 부식 이상이 `corrosion` 상태 부착(아델리아 외 부식 소스도 소모 가능) + `vuln:all` 만료 시 `corrosion` 동반 제거. 검증: 부식 소모 → 물리/아츠 취약 0.12·부식 해제 ✅
- ⚠ 단순화: 마운틴 서퍼(부식 다수 시 배틀 재발동), 친구의 그림자 그림자 회수 메커니즘(즉시 회복으로 근사), 궁 확률 생성(절반 근사), 잠재.

### 🔑 부식 셋업→소모 사이클
연계 화산 분화(부식 부여) → 다음 턴 배틀 질주하는 돌리(부식 소모 → 물리+아츠 취약 30초). 아츠 부착을 안 써서 부착 누적형 딜러(레바테인 흡수·라스트 라이트 소모)와 스택 충돌 없음 = 아델리아 정체성. 첫 힐(`healUnit`)·`corrosion` 마커는 향후 서포터(자이히·안탈)·캐스터(플루라이트 자연)에 재사용.
