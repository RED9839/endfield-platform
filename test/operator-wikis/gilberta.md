# 질베르타 (Gilberta) — 자연 / 아츠 유닛 / 서포터 ★6 (한정)

주스탯 **의지**, 보조 **지능**. 궁충 효율 + 광역 아츠 취약(게임 내 최고) + 몹몰이 + 강제 띄우기. 서포터 1황.

## 스킬
- **기본 비전 지팡이·에너지 제어**: 4단 자연. 강일 16 불균형.
- **배틀 비전 지팡이·중력 모드** (게이지 100): 중력 특이점(몹몰이, 인력 97%→219%) → 시전 끝 폭발 58%→130% 자연 + **자연 부착**. 폭발 불균형 10.
- **연계 비전 지팡이·매트릭스 이동** (쿨 20~19초): **아츠 이상 부여 적 있을 때**. 목표+주변 모든 적 중력 끌어당김(몹몰이), 140%→315% 자연 + **강제 띄우기**(= 방어 불능 1스택, 띄우기 1회). 불균형 5.
- **궁극 비전 지팡이·중력장** (게이지 90): 5초 중력 혼란 구역. 333%→750% 자연 + 자연 부착 + **감속 80% + 아츠 취약 18~30%**(방어 불능 스택당 추가 1.75~3%). 불균형 20.

## 재능
- **전달자의 노래**: 필드에 있으면 팀 가드/캐스터/서포터 **궁극기 충전 효율 +7%**.
- **뒤늦은 편지**: 배틀 마지막타 or 연계가 2명+ 명중 시 메인 **[72~108 + 지능×0.6~0.9]** 회복(만피면 최저 HP 아군).

## 🎮 현재 구현
- skillMechanic `combo-strike`(배틀/궁 자연 부착 자동)
- passive: `{ teamUltPct: 0.20, healOnCast: 10, linkLaunch: true }`
  - `teamUltPct 0.20` = 전달자의 노래(궁충 효율, 카드 게이지 경제에 맞춰 상향) / `healOnCast 10` = 뒤늦은 편지(장비 등급 비례)
  - `linkLaunch` = 매트릭스 이동 강제 띄우기 = **방어 불능 1스택(단발, 누적·페이오프 없음)**. 본인은 띄우기 딜러 아니라 스택 깔아 아군 강타/쇄빙 보조
- ULT_KIND **zone**(중력 구역 취약 0.25), ULT_ENERGY 90. 몹몰이(위치)는 미모델
