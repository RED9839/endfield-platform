# 질베르타 (Gilberta / 전작 안젤리나의 리컨비너) — 자연 / 아츠 유닛 / 서포터 ★6 (한정)

**서포터 1황**. **인게임 최고 광역 아츠 취약궁(방불 비례)** + **유일 배틀 몹몰이** + 강제 띄우기 연계(방불) + 가드/캐스터/서포터 궁충 효율 재능. 누킹형 딜러(이본·로시·레바테인) 코어 파츠. "빈자리 나면 일단 질베르타 넣으면 반은 간다". 주스탯 **의지**, 보조 **지능**. 불포(여우, 로도스).

## 스킬 (설명문 원문)

### 기본 공격 「비전 지팡이 · 에너지 제어 기술」
최대 4단 자연. 강력한 일격 16 불균형. **처형 400→900%**.

### 배틀 스킬 「비전 지팡이 · 중력 모드」 (게이지 100)
> 지속 시전, 전방 중력 특이점 생성. 주변 적 **끌어당김(몹몰이)** + 자연. 시전 끝 폭발 → 자연 + **자연 부착**.
- 인력 97→219% / 폭발 58→130% / 폭발 불균형 10

### 연계 스킬 「비전 지팡이 · 매트릭스 이동」 (쿨 20~19초)
> **아츠 이상 상태 부여한 적이 있을 때**. 목표+주변 모든 적 중력 끌어당김 + 자연 + **강제 띄우기**.
- 140→315% / 불균형 5 / 쿨 20→19초

### 궁극기 「비전 지팡이 · 중력장」 (게이지 90, 쿨 20초)
> 5초 중력 혼란 구역. 즉시 자연 + 자연 부착. **감속 + 아츠 취약**. 목표가 **방어 불능이면 아츠 취약이 방불 스택 비례 추가**. 띄우기 적은 구역 종료까지 띄우기 유지.
- 333→750% / 기초 아츠 취약 18→30% / 방불 1스택당 +1.75~3% / 감속 80% / 불균형 20

## 재능 (설명문 원문)
- **전달자의 노래**(1정): 필드에 있을 때 팀 **가드/캐스터/서포터 궁극기 충전 효율 +4%(1정)→+7%(2정)**.
- **뒤늦은 편지**(2정): 배틀 마지막 공격 또는 연계가 **2명 이상 명중 시** 메인 **[72+지능×0.6](2정)→[108+지능×0.9](3정)** 회복. 만피면 최저 HP% 아군.
- 장비 조합 I/II/III(파랑/보라/노랑).

## 🎮 현재 구현 (DD류 — `app/game/dd/roster.ts` gilberta, Lv1 계수)
- OP_BASE: cls **supporter**, hp 2689, attack 110, speed 62, ultCost **90**. 주스탯 의지.
- **중력 모드**(배틀, power 1.55=인력+폭발, `target:all`, `attach:nature`) — 몹몰이 광역 자연 부착 + 뒤늦은 편지. 검증: 자연 부착·회복 ✅
- **매트릭스 이동**(연계, power 1.40, `target:all`, cooldown 4, `anomaly:launch`, `requires` 아츠 이상 적) — 광역 강제 띄우기(방불). 검증: 게이트 false→(연소 후)true ✅
- **중력장**(궁, power 3.33, `target:all`, `attach:nature`, ultCost 90) — `apply`: **아츠 취약 0.18+방불×0.0175**(방불 비례, 5초≈1턴) + **감속**(speedMod -30). 검증: 방불2 → 아츠취약 0.215 ✅
- 엔진: **전달자의 노래**(배틀 궁충 분배 시 가드/캐스터/서포터 ×1.07), **`combustion` 상태 마커**(연소도 태그 → 아츠 이상 감지 통일: frozen/dot/shock/corrosion/combustion), 뒤늦은 편지 회복(2+ 명중). 검증: 미브 궁충 ×1.07 ✅

### 🔑 아츠 취약 위계 + 방불 시너지
질베르타 궁의 **방불 스택 비례 아츠 취약**(기초 0.18 + 방불×0.0175)이 핵심 — 엠버/카치르/진천우가 쌓은 방불을 취약으로 환산해 누킹 딜러(이본·로시) 폭딜 증폭. 연계 강제 띄우기로 본인도 방불 1스택 공급. 자연 부착이 비자연팟 부착을 이상으로 전환하는 역시너지는 위키 명시(라스트 라이트 냉기팟 주의).
