# 관리자 (Endministrator) — 물리 / 한손검 / 가드 ★6 (주인공)

주스탯 **민첩**, 보조 **힘**. 가드의 탈을 쓴 물리 스트라이커 — 방어 불능 강타 + 오리지늄 결정 부착→파괴 폭딜. 본인은 방어 불능 부여 수단 없음(아군 의존).

---
## 스킬 (설명문 원문)

### 기본 공격 「훼손 시퀀스」
적에게 최대 5단 공격, 물리 피해. 메인 컨트롤 시 강력한 일격 18 불균형.
- 배율(Lv1→12): 1단 23%→51% … 5단 40%→90% / 낙하 80%→180% / 처형 400%→900%

### 배틀 스킬 「구성 시퀀스」 (게이지 100)
> 오리지늄 아츠를 사용해 전방 일정 범위 내의 적을 공격하여 물리 피해를 주고 **강타**합니다.
- 배율 156%→350%, 불균형치 10. **AoE(부채꼴)**
- ⚠ **강타 = 방어 불능 스택 소모 → 대량 물리.** 표시 배율은 발동 피해일 뿐, **강타 피해는 배틀 배율이 아니라 아츠 강도(공격력)에 비례**. 강타 유발하려면 미리 방어 불능 스택 누적 필요 → **조건부: 스택 0이면 강타 미발동 + 자가 생성 X**(관리자는 전무한 방어 불능 부여 → 아군 빌더 의존)

### 연계 스킬 「봉인 시퀀스」 (쿨 16~15초, 조건: 아군 연계 피해 시)
> 적의 근처로 돌진하여 물리 피해 + **오리지늄 결정 부착** + 일정 시간 봉인. 물리 이상/방어 불능 부여하면 결정 소모 + 추가 물리.
- 배율 45%→100% / 결정 파괴 178%→400% / 불균형 10 / 봉인 4→5초
- 잔몹만 봉인, 엘리트/보스는 봉인 안 걸림(결정 부착은 동일). 재능 본질 붕괴·현실 정지가 이 결정에 연동.

### 궁극기 「폭격 시퀀스」 (게이지 80)
> 지면 강타, 전방 부채꼴 대량 물리. **결정 부착 적이면 결정 파괴 + 1회 추가 물리.**
- 배율 356%→800% / 추가 480~600% / 불균형 25. **결정 부착 기준 ~1500% 누킹**

## 재능 (설명문 원문)
- **본질 붕괴**: 적에게 부착한 오리지늄 결정이 소모됐을 때, 자신의 공격력 **+30%**, 15초 지속(중첩 X).
- **현실 정지**: 오리지늄 결정이 부착된 적이 받는 물리 피해 **+20%**.

## 잠재능력
- 01 마지막 각성: 배틀 구성 시퀀스가 결정 소모 시 스킬 게이지 50 반환.
- 02 권능 반영: 본질 붕괴 강화 — 공격력 증가 시 다른 아군에게 절반(15%) 부여.
- 03 길잡이: 궁극이 결정 소모 후 궁극 에너지 15 획득.

## 🎮 현재 구현 (DD류 리뉴얼 — `app/game/dd/`)

카드/드로우 폐기 → **속도 턴순서 + 포지션 + 스킬 사용게이트(usage gate)**. 카드 모델에서 깨지던 "연계 조건"이 DD류에선 자연 흡수됨.

스킬킷 (`roster.ts` endministrator, element 물리, 불균형치 명시):
- **구성 시퀀스**(배틀 156%=1.56, 불균형 10, `anomaly:"crush"`) — 방불 소모 강타(300/450/600/750%). 연타 미적용.
- **봉인 시퀀스**(연계 45%=0.45, 불균형 10, `crystal:true`, `requires: lastLinkAlly && !==self`) — **아군 연계 후에만**(게이트), 결정 부착·봉인. 자체 방불 부여 없음.
- **폭격 시퀀스**(궁 356%=3.56, 불균형 25, selfUlt) — 광역 강타 + 결정 파괴 추가 물리(267%).

엔진 처리 (`combat.ts`):
- **결정 소모**: 결정 부착 적에게 물리 이상/궁극 명중 시 → 결정 파괴 추가 물리 + **본질 붕괴**(`adm.atkBuff=0.30`). 배율 **구분**: 물리 이상 파괴 = 연계 배율 **178%**, 궁 파괴 = 궁 추가 배율 **267%**. **다른 오퍼 스킬로도 깨짐**(위키 정합) — `s.units.find(endministrator)`로 관리자 귀속.
- **현실 정지**: act에서 결정 부착 적 **물리 한정** ×1.2(+20%) — `hadCrystal && elem==="physical"`.
- **본질 붕괴**: `eb()`(유효 공격력)에 atkBuff 반영, 모든 피해에 적용.
- **연계 게이트**: 아군 link 사용 시 `s.lastLinkAlly` → 관리자 봉인 unlock.

검증(`tsx test/dd.ts` 보스 R4): 진천우 띄우기→관리자 봉인(결정 부착, 자기소비 X)→여풍 넘어뜨리기가 결정 파괴(178%)+본질 붕괴→포그 갑옷파괴 3스택 200%. 일반전 3R·보스 6R 승리. ✅

미구현(후속): 기본 공격 5단/처형(불균형 적 처형+게이지 회복) — DD 모델에 기본공격 추가 필요. atkBuff 지속시간(15초)은 전투 내내 유지로 단순화. 잠재(마지막 각성·권능 반영·길잡이)는 명함 baseline이라 제외.
