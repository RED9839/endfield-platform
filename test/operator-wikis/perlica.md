# 펠리카 (Perlica) — 전기 / 아츠 유닛 / 캐스터 ★5

**첫 캐스터 · 타이틀 히로인(아미야 포지션)**. 즉발 **전기 부착** + 쉬운 조건 **강제 감전 연계(아츠 취약)** + **깡딜 궁** + 불균형 추가딜. 연계 운용 부담 적고 조건 안 타 전기팟 외 아츠 조합에도 범용 공격 서포터. 1.2 장방이 출시 후 전기 전담 서폿. 주스탯 **지능**, 보조 **의지**. 리베리(엔드필드 공업 감독관).

## 스킬 (설명문 원문)

### 기본 공격 「프로토콜α · 돌파」
최대 4단 전기. 강력한 일격 15 불균형. **처형 400→900%**.

### 배틀 스킬 「프로토콜ω · 뇌격」 (게이지 100)
> 하늘에서 전기 에너지 떨어뜨려 좁은 범위 전기 + **전기 부착**.
- 178→400% / 불균형 10 / 거의 즉발

### 연계 스킬 「실시간 프로토콜 · 연쇄 섬광」 (쿨 20~19초)
> **메인 컨트롤 오퍼레이터가 강력한 일격 피해를 준 다음**. 누적 전기 에너지 방출 전기 + **5초 강제 감전**(감전 = 받는 아츠 피해↑, 1레벨 12%).
- 80→180% / 불균형 10 / 쿨 20→19초

### 궁극기 「프로토콜ε · 70.41κ」 (게이지 80, 쿨 10초)
> 즉시 궤도 부유물 전송 → 구역 내 적 대량 **전기 깡딜**.
- 445→1000% / 불균형 20

- **오블리터레이션 프로토콜**(1정): **불균형 상태 적**에게 주는 피해 +20%(1정)→+30%(2정).

## 🎮 현재 구현 (DD류 — `app/game/dd/roster.ts` perlica, Lv1 계수)
- OP_BASE: cls **caster**(첫 캐스터), hp 2689, attack 110, speed 66, ultCost **80**. 주스탯 지능.
- **프로토콜ω·뇌격**(배틀, power 1.78, `attach:electric`) — 즉발 전기 부착. 검증: 전기 부착 1 ✅
- **연쇄 섬광**(연계, power 0.8, cooldown 4, `forceShock:true`) — **강제 감전**(부착 무관 shock + 아츠 취약 0.12). 검증: 감전·아츠취약 0.12 ✅
- **프로토콜ε·70.41κ**(궁, power 4.45, `target:all`, ultCost 80) — 광역 깡딜 전기. 검증: 일반 548 → 불균형 927(×1.69) ✅
- 엔진: **오블리터레이션 프로토콜**(`perlica && staggered → ×1.3`, 전역 불균형 ×1.3과 별도 → 총 ×1.69). `forceShock`(아크라이트와 공유, 부착 무관 직접 감전).
- ⚠ 단순화: 순환 프로토콜(연계 방불 재튕김), 좁은 범위(단일), 감전 레벨(고정 1=12%), 잠재.

### 🔑 강제 감전 = 부착 무관 아츠 취약
일반 감전은 전기 부착→다른 속성 부착으로 아츠 이상 발생이 필요하지만, 펠리카 연계 `forceShock`는 **부착 없이 직접 감전 + 아츠 취약 0.12** 부여 → 아무 아츠 딜러에게나 취약 셋업 가능(범용성의 근거). 향후 장방이(감전 소모/레벨업) 연동 대비. 첫 캐스터로 가드/캐스터/서포터 궁충 재능(질베르타) 수혜 직군에 포함.
