# 펠리카 (Perlica) — 전기 / 아츠 유닛 / 캐스터 ★5

주스탯 **지능**, 보조 **의지**. 전기 부착 + 강제 감전(아츠 취약) + 깡딜 궁. 전기팟 전담 서폿(아미야 포지션 타이틀 히로인).

## 스킬
- **기본 프로토콜α·돌파**: 4단 전기. 강일 15 불균형.
- **배틀 프로토콜ω·뇌격** (게이지 100): 좁은 범위 178%→400% 전기 + **전기 부착**(거의 즉발). 불균형 10.
- **연계 실시간 프로토콜·연쇄 섬광** (쿨 20~19초): **메인 강일 후**. 80%→180% 전기 + **5초 강제 감전**(감전 = 받는 아츠 피해↑, 1레벨 12%). 불균형 10.
- **궁극 프로토콜ε·70.41κ** (게이지 80): 궤도 부유물 즉시 광역 **445%→1000% 전기 깡딜**. 불균형 20.

## 재능
- **오블리터레이션 프로토콜**: 불균형 상태 적에게 피해 +20~30%.
- **순환 프로토콜**: 연계가 방어 불능 적 명중 시 1회 더 튕김.

## 🎮 현재 구현
- skillMechanic `electric-attachment`(배틀 전기 부착)
- passive: `{ vsBroken: 0.20, linkForceShock: 0.12 }`
  - `vsBroken 0.20` = 오블리터레이션(불균형 적 특화)
  - `linkForceShock 0.12` = 연쇄 섬광 강제 감전(연계 → 감전 상태 + 아츠 취약 0.12, 전기 단일로도. useRunState 링크 훅)
- ULT_ENERGY 80. 감전 = artsVuln(아츠 전용 취약, getStatusBonus). 순환(연계 재튕김)은 미모델
- ※ vsBroken은 한때 0.30으로 올렸다가 범위 외라 0.20 복귀
