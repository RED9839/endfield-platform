# 판 (Da Pan / 본명 판 원) — 물리 / 양손검 / 스트라이커 ★5

**유일 비6성 물리 스트라이커**(관리자 외). **방어 불능 4스택 강타 단발 누커** — 팀이 쌓은 방불을 연계 강타로 소모해 한방딜. 띄우기/넘어뜨리기로 방불도 직접 빌드. 재능·전무 전부 연계 강타 화력 몰빵 → 단발 대미지는 6성급. 주스탯 **힘**, 보조 **의지**. 우르수스(홍산 과학원 전 천사·요리사). 순수 물리팟의 한 축.

---
## 스킬 (설명문 원문)

### 기본 공격 「돌려가며 썰기!」
최대 4단 물리. 강력한 일격 20 불균형. **처형 400→900%**.

### 배틀 스킬 「뒤집어 주지!」 (게이지 100)
> 웍 차지 후 위로 던져 물리 + **띄우기**. (5돌: 단일 명중 시 방불 2스택)
- 133→300% / 불균형 10 / 에어본 1.8초

### 연계 스킬 「조미료 뿌리기!」 (쿨 20~19초)
> **방어 불능 4스택** 적일 때. 웍 대량 물리 + **강타**. 이번 강타 피해 +10%.
- 289→650% / 추가 강타 피해 +10%(Lv9+ 15%·Lv12 20%) / 불균형 15 / SP 무소모

### 궁극기 「채 썰어 웍에 넣기!」 (게이지 90, 쿨 15초)
> 도마 내리쳐 전방 강제 **띄우기** + 6단 베기 + 마지막 추락 강제 **넘어뜨리기** + 대량 물리. (방불 2스택 빠른 적립)
- 공중 6단 22→50%(×6) / 궁 178→400% / 불균형

## 재능 (설명문 원문)
- **전분 풀기**(1정): 방불 1스택 소모마다 물리 피해 **+4%(1정)→+6%(2정)**, 10초, 최대 4스택(=+24%).
- **간 맞추기**(2정): 궁 마지막 명중 시 식재료 준비 스택(최대 1→2). 그 상태에서 연계 명중 시 **쿨 즉시 40% 회복** + 스택 소모.
- 장비 조합 I/II/III(파랑/보라/노랑).

## 잠재능력 (명함 제외)
01 정교한 요리(궁 처치 시 물리 +30%) / 02 맛의 조화(간 맞추기 +10초·스택 +1) / 03 모범 직원(힘 +15·물리 +8%) / 04 자체 제작 소스(궁 에너지 -15%) / 05 센 불로 소스 졸이기(배틀 단일 명중 시 방불 +1, 45초마다)

## 🎮 현재 구현 (DD류 — `app/game/dd/roster.ts` dapan, Lv1 계수)
- OP_BASE: cls **striker**, hp 2689, attack 110, speed 55, ultCost **90**. 주스탯 힘.
- **뒤집어 주지!**(배틀, power 1.33, `anomaly:launch`) — 물리 + 띄우기(방불 빌더).
- **조미료 뿌리기!**(연계, power 2.89, cooldown 4, `anomaly:crush`, `crushAmp:1.1`, `requires` 방불 4스택) — 대량 물리 + **강타(방불 4스택 소모 ×1.1)** + id훅 **전분 풀기**(소모 스택×6% 물리 증폭, 최대 0.24). 검증: 방불 4 소모·물리증폭 0.24·×1.1 강타 ✅
- **채 썰어 웍에 넣기!**(궁, power 3.10, `target:all`, `anomaly:launch` + apply 방불 +1, ultCost 90) — 광역 물리 + **방불 2스택**(띄우기+넘어뜨리기). 검증: 적 방불 2 ✅
- 엔진 신규: **`crushAmp`**(강타 payoff 추가 배율, applyAnomaly에서 적용), **전분 풀기**(`primaryPre`로 소모 스택 캡처 → self amp.physical 누적).
- ⚠ 단순화: 간 맞추기(궁→연계 쿨 회복), 5돌 방불 2스택, 잠재. 관리자(`endministrator`)와 강타 메커니즘 공유.

### 🔑 방불 4스택 강타 사이클 (관리자 유사·고배율)
판/여풍/진천우로 방불 빌드 → 포그 갑옷 파괴 → 다시 4스택 → **판 연계 강타(650%+×1.1)로 단발 폭딜**. 관리자 구성 시퀀스(강타)와 동일 메커니즘이나 **연계라 SP 무소모 + 추가 강타 +10% + 고배율** → 단발 누킹은 관리자 상회. 본인 궁으로 방불 2스택 빠르게 적립 가능. `crushAmp`는 향후 강타 강화 오퍼에 재사용.
