# 자이히 (Xaihi) — 냉기 / 아츠 유닛 / 서포터 ★5

주스탯 **의지**, 보조 **지능**. 퓨어 힐러 + 아츠 증폭 + 냉기 부착/취약 + 냉기/자연 증폭궁. (냉기팟 핵심, "7성")

## 스킬
- **기본**: 5단 냉기. 강일 15 불균형.
- **배틀 디도스** (게이지 100): 지원 결정체 20초. 메인이 강일 시 **치유**(144~324 + 의지×0.34~0.76, 최대 2회). **오버힐(메인 풀피)이면 아츠 증폭**(9~15%, 25초).
- **연계 스트레스 테스트** (쿨 8~7초): **결정체 회복 횟수 소진 시**. 200%→450% 냉기 + **냉기 부착**. 불균형 10.
- **궁극 스택 오버플로** (게이지 80): 팀 전체 **냉기 증폭 + 자연 증폭**(11~24% + 지능×, 최대 +30%, 12초).

## 재능
- **가동 프로세스**: 연계 명중 시 냉기 부착/동결 적이면 받는 냉기 +10%(5초).
- **프리징 프로토콜**: 궁이 팀 냉기 부착/동결 정화.

## 🎮 현재 구현
- skillMechanic `combo-strike`(연계 냉기 부착 자동)
- passive: `{ targetVuln: 0.12, healOnCast: 14, teamAmp: 0.05 }`
  - `healOnCast 14` = 디도스 퓨어 힐러 치유(빠져 있던 핵심, 의지=장비 등급 비례)
  - `teamAmp 0.05` = 디도스 아츠 증폭. ⚠ 0.10은 sim 47→42% 붕괴(kill-timing) → 0.05로 고정
  - `targetVuln 0.12` = 가동 프로세스 냉기 취약
- ULT_KIND **buff**(dmgBuffPct, 냉기/자연 증폭), ULT_ENERGY 80. 프리징 정화 미모델
