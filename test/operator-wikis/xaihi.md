# 자이히 (Xaihi / 세라핀 자이히) — 냉기 / 아츠 유닛 / 서포터 ★5

**"냉기 파티의 꽃 · 탈 5성 개사기 · 7성"**. 혼자서 **퓨어 힐 + 아츠 증폭 + 냉기/자연 증폭 + 냉기 부착**을 전부 부여. 5성이라 한돌 쉬움·튜토리얼 담당으로 접근성 최상. 주스탯 **의지**, 보조 **지능**(반대 배치). 살카즈(고요한 수도회, 라테라노 유래). 강력한 일격 트리거(메인 의존)가 유일 단점.

---
## 스킬 (설명문 원문)

### 기본 공격 「쿨타임」
최대 5단 냉기. 강력한 일격 15 불균형. **처형 400→900%**.

### 배틀 스킬 「디도스」 (게이지 100)
> 지원 결정체 소환 20초 메인 추적. 메인이 강력한 일격 시 **치유**(최대 2회, 의지→치유량). 회복 시 메인이 **최대 체력(오버힐)이면 25초 아츠 증폭** 부여(비중첩).
- 기초 치유 144→324 / 의지당 0.34→0.76 / 아츠 증폭 9→15%

### 연계 스킬 「스트레스 테스트」 (쿨 8~7초)
> 지원 결정체 **회복 횟수 모두 소모** 시. 결정체 투척 냉기 + **냉기 부착**.
- 200→450% / 불균형 10 / 쿨 8→7초

### 궁극기 「스택 오버플로」 (게이지 80, 쿨 20초)
> 팀 전체 12초 **냉기 증폭 + 자연 증폭**. 지능이 증폭 추가 강화(최대 +30%).
- 기초 11→24% / 지능당 0.014→0.03% / 지능 상한 30%(M3 36%)

## 재능 (설명문 원문)
- **가동 프로세스**(1정): 연계 명중 시 목표가 **냉기 부착/동결** 상태면 받는 냉기 +7%(1정)→+10%(2정), 5초.
- **프리징 프로토콜**(3정): 궁이 추가로 팀 전체 냉기 부착/동결 정화.
- 장비 조합 I/II/III(파랑/보라/노랑).

## 잠재능력 (명함 제외)
01 애자일 엑스큐션(디도스 아츠 증폭 +5%) / 02 링크 어그리게이션(궁 에너지 -10%) / 03 노드 매핑(연계 주변 1명 추가 명중) / 04 그레이스케일 릴리스(지능 +15·치유 효율 +10%) / 05 리커전 컨트롤(궁 증폭 ×1.1)

## 🎮 현재 구현 (DD류 — `app/game/dd/roster.ts` xaihi, Lv1 계수)
- OP_BASE: cls **supporter**, hp 2689, attack 110, speed 64, ultCost **80**. 주스탯 의지.
- **디도스**(배틀, `target:self`, power 0) — id훅: 부상 메인 **치유**(`144+gearGrade×0.34`=164) / **오버힐(전원 만피)이면** 최전열 아군에 **아츠 증폭 0.09**(25초≈5턴). `timers.didos`로 연계 활성. 검증: 회복+164·오버힐 아츠 증폭 0.09 ✅
- **스트레스 테스트**(연계, power 2.0, `attach:cryo`, cooldown 2, `requires` 디도스 활성) — 냉기 부착 + `apply`: **가동 프로세스**(냉기/동결 적 → 냉기 취약 0.10, 5초≈1턴). 검증: 게이트·부착·냉기취약 0.10 ✅
- **스택 오버플로**(궁, `target:self`, ultCost 80) — id훅: 팀 전체 **냉기 증폭 + 자연 증폭**(`0.11+min(0.30, gearGrade×0.003)`=0.29, 12초≈3턴). 검증: 냉기/자연 증폭 0.29 ✅
- 엔진: 디도스 오버힐 분기(전원 만피 → 아츠 증폭), 궁 이중 속성 증폭(`amp.cryo`+`amp.nature`). 첫 힐(`healUnit`) 재사용.
- ⚠ 단순화: 강력한 일격 2회 트리거(디도스 즉시 치유로 근사), 회복 2회 제한, 지능→`gearGrade` 치환, 프리징 프로토콜 정화(아군 냉기 부착 없어 휴면), 잠재.

### 🔑 오버힐 → 아츠 증폭 분기
디도스가 **부상 아군이 있으면 치유**, **전원 만피면(오버힐) 메인에 아츠 증폭**으로 자동 분기 = 자이히 핵심 메커니즘. 체력 유지가 곧 증폭 유지. 향후 안탈(조건부 치유/증폭)에 패턴 재사용.
