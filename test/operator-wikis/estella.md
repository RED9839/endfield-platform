# 에스텔라 (Estella) — 냉기 / 장병기 / 가드 ★4

주스탯 **의지**, 보조 **힘**. 냉기 부착 + 동결→쇄빙(강제 띄우기) + 물리취약 + 방불. **동결 의존**(자체 동결 불가 → 알레쉬·이본·질베르타 등 동결/부착 파트너 필요). 4성이라 접근성 높음. 물리/쇄빙 파티 서포터.

---
## 스킬 (설명문 원문)

### 기본 공격 「노이즈」
적에게 최대 4단 물리. 메인 컨트롤 시 강력한 일격 17 불균형.
- 배율(Lv1→12): 1단 25%→56% … 4단 40%→90% / 낙하 80→180% / 처형 400→900%

### 배틀 스킬 「서스테인」 (게이지 100)
> 전방으로 장창을 찔러 냉기 음파 방출. 일직선상의 적에게 냉기 피해 + **냉기 부착**.
- 배율 150%→350% / 불균형 10. **일직선 AoE**

### 연계 스킬 「디스토션」 (쿨 18~17초, 조건: 적이 동결 상태)
> 빠르게 접근, 좁은 범위 물리 + **강제 띄우기**. 동결 적 명중 시 추가 피해 + 6초 **물리 취약**.
- 비동결 160%→360% / **동결 280%→630%** / 물취 10%→15%. **강제 띄우기 → 동결 적이면 쇄빙**

### 궁극기 「트레몰로」 (게이지 70)
> 창 내려찍기, 원형 AoE 물리. 적이 **물리 취약이면 강제 띄우기**.
- 배율 489%→1100% / 불균형 15. 게이지 70(저렴)

---
## 재능 (설명문 원문)
- **공감**: 쇄빙 발동마다 다음 서스테인이 게이지 +7.5(1차)/**+15(2차)** 반환(중첩 X).
- **이유 있는 게으름**: 냉기 부착 면역 + 받는 냉기 -10%(2차)/**-20%(3차)**. (위기협약 메인오퍼 동결 제약 카운터)
- 능력치 상승: 의지 +10/+15/+15/+20.

## 잠재능력 (명함 제외)
- 01 습관성 지각(디스토션 물취 +3초) / 02 낮은 기대치(궁 에너지 -10% → 63) / 03 작업 연기(서스테인 거리·첫타 +40%) / 05 생존이 승리다(동결 부여마다 궁 에너지 +5)

## 🎮 현재 구현 (DD류 — `app/game/dd/roster.ts` estella, Lv1 계수)
- OP_BASE: cls guard, hp 2689, attack 110(튜닝), speed **47**(민첩 낮음), ultCost **70**.
- **서스테인**(배틀, power 1.5, `element:cryo`, `attach:cryo`, target row) — 냉기 부착.
- **디스토션**(연계, power 2.8=동결적 280%, `element:physical`, `anomaly:launch`, `requires: t.frozen>0`, cooldown 4턴, `apply` 물리취약 0.10) — **강제 띄우기 → 동결 적이면 엔진 `tryShatter`로 쇄빙**. 검증: 동결 전 사용불가 → 동결 후 쇄빙 360%+띄우기 ✅
- **트레몰로**(궁, power 4.89, target all, `anomaly:launch`, selfUlt, ultCost 70) — 원형 AoE + 강제 띄우기.
- 미구현: **공감**(쇄빙 게이지 반환 — tryShatter→state 배선 필요), **이유 있는 게으름**(냉기 면역 — 아군 부착 미모델), 기본공격 처형, 잠재.
- ⚠ **동결 파트너 필요**: 에스텔라 단독은 냉기만 부착 → 동결 불가 → 디스토션/쇄빙 미발동. 동결 소스(알레쉬 등) 이식 후 풀 콤보 테스트 예정.
