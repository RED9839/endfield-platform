# 에스텔라 (Estella) — 냉기 / 가드 ★?

※ 위키 원문 배율표는 미보유(구현 기반 복원). 냉기 부착 + 동결 활용 쇄빙 + 물리 띄우기 하이브리드. physBreak **build**.

## 스킬
- **배틀 서스테인** (일직선 AoE): 냉기 음파 일직선 냉기 + **냉기 부착**.
- **연계 디스토션** (cond: 동결 적, AoE): 접근 좁은 범위 물리 + **강제 띄우기**. 동결 적 명중 시 추가 효과(쇄빙).
- **궁극 트레몰로** (원형 AoE): 창 내려찍기 물리. **물리 취약 적이면 강제 띄우기**.

## 재능
- **공감**: 쇄빙 발동 시 게이지(전투 에너지) 반환.
- **이유 있는 게으름**: 냉기 면역, 받는 냉기 -20%.
- **생존이 승리다**: 동결 부여 시 궁충.
- (디스토션 물리 취약)

## 🎮 현재 구현
- passive: `{ shatterEnergy: 1, damageResist: 0.10, ultOnFreeze: 5, linkPhysVuln: 0.12 }`
  - `shatterEnergy 1` = 공감(쇄빙 게이지 반환) / `damageResist 0.10` = 이유 있는 게으름
  - `ultOnFreeze 5` = 생존이 승리다(동결 시 궁충) / `linkPhysVuln 0.12` = 디스토션(연계 물리 취약, 원소 무관)
- physBreak build(+물리 이상 게이트 확장으로 cryo 하이브리드 띄우기 가능). ULT_ENERGY 70
- ※ 부식은 물리·아츠 둘 다 적용(corrodeVuln, 위키 예외) — 한때 arts-only로 잘못 구현했다가 정정
