# 엠버 (Ember / 본명 아즈릴라) — 열기 / 양손검 / 디펜더 ★6

**첫 6성 디펜더**(1.2 기준 유일 6성 디펜더). "열기 속성의 탈을 쓴 **물리 디펜더**" — 넘어뜨리기로 방불·불균형 누적 + 치유·비호·팀 보호막. 주스탯 **힘**, 보조 **의지**(→ 높은 체력·치유 스케일). 산크타(철의 서약군). 느린 공속·수동적 피격 운용 탓에 6성 기피 1순위지만, 위기 협약·회피 불가 제약에선 사실상 필수.
※ 열기 부착/연소 효과 전무(각주6), 배틀·궁만 열기 피해.

---
## 스킬 (설명문 원문)

### 기본 공격 「돌진 검술」
최대 4단 물리. 강력한 일격 25 불균형. 낙하 공격(공중). **처형 400→900%**(불균형 적 + 게이지 회복).

### 배틀 스킬 「진군」 (게이지 100)
> 전방으로 뛰어올라 내리찍어 **부채꼴 범위 모든 적**에게 **열기 피해** + **넘어뜨리기**. 시전 도중 공격받으면 명중 시 **추가 불균형치**.
- 173→390% / 불균형 10 / **공격받은 후 추가 불균형 10**

### 연계 스킬 「전선에서의 지원」 (쿨 19~18초)
> **메인 컨트롤 오퍼레이터가 공격받았을 때** 사용 가능. 뛰어올라 내리찍어 물리 + **넘어뜨리기** + **메인 컨트롤 오퍼레이터 치유**. **의지**가 치유량 추가 증가.
- 102→230% / 기초 치유 300→675 / 의지 1당 0.7→1.58 / 불균형 10 / 쿨 19→18초

### 궁극기 「다시 불타오르는 맹세」 (게이지 100, 쿨 20초)
> 지면 강타 주변 적 **열기 피해** + 10초간 **팀 전체에 엠버 최대 생명력 비례 보호(보호막)**.
- 289→650% / 보호 전환 비율 18→25%(생명력) / 불균형 25

## 재능 (설명문 원문)
- **전진의 결의**(1정): 진군·전선에서의 지원 발동 중 **30%(1정)→50%(2정) 비호** + 스킬 끊김 방지.
- **강철에는 강철로**(2정): 적에게 피해받을 때 **공격력 +6%(2정)→+9%(3정)**, 7초, **최대 3스택**.
- 인프라: 제조실 경험치 효율↑ / 응접실 단서4(철의 서약군). 장비 조합 I/II/III(파랑/보라/노랑).

## 잠재능력 (명함 제외)
01 이동 요새(비호 +20%·명중 시 +1.5초) / 02 무쇠의 전사(힘·의지 +20) / 03 불굴의 전선(연계가 최저 체력% 1명 추가 치유 50%) / 04 불멸의 불꽃(궁 에너지 -15%) / 05 철의 서약(궁 보호 ×1.2 + 보호 대상 공격력 +10%)

## 🎮 현재 구현 (DD류 — `app/game/dd/roster.ts` ember, Lv1 계수)
- OP_BASE: cls **defender**(첫 디펜더), hp 2689, attack 110, speed **40**(느린 공속, 전 오퍼 최하위), ultCost **100**.
- **진군**(배틀, power 1.73, `element:heat`, `target:row`, `anomaly:knockdown`) — 부채꼴 열기 + 넘어뜨리기. **전진의 결의** 50% 비호.
- **전선에서의 지원**(연계, power 1.02, `requires st.allyHit`(피격 트리거), cooldown 4, `anomaly:knockdown`) — 넘어뜨리기 + **최저 체력% 아군 치유**(`300+gearGrade×0.7`=342) + 비호. 검증: 회복+342·방불+1·비호0.5 ✅
- **다시 불타오르는 맹세**(궁, power 2.89, `element:heat`, `target:row`, ultCost 100) — 광역 열기 + **팀 전체 보호막**(`maxHp×0.18`=484, 2턴). 검증: 팀 보호막 484 ✅
- 엔진 신규: **`s.allyHit`(피격 트리거 윈도우)** — 아군 피격 시 set, `startRound`에서 리셋 → 엠버 연계 게이트. **강철에는 강철로**(엠버 피격 시 atkBuff +0.09 최대 0.27). **전진의 결의**(배틀/연계 시 protection 0.5, 1턴). 궁 팀 보호막(`applyBuff shield`).
- ⚠ 단순화: 진군 "시전 중 피격 추가 불균형"(DD에 중간 캔슬 없음), 의지→치유 스케일은 `gearGrade` 치환, 기본공격 4단/낙하/처형, 잠재(03 추가 치유·05 보호 강화).

### 🔑 피격 트리거(`s.allyHit`)
엠버의 정체성인 "메인 피격 시 연계" 수동 운용을 충실 모델. 적이 아군에 유효타 → `allyHit=true`(라운드 한정). 엠버 본인 피격 시 `강철에는 강철로` 동시 발동. `healUnit`(카뮤에서 도입)에 자동 연동.
