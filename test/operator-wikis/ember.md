# 엠버 (Ember) — 열기 / 양손검 / 디펜더 ★6

주스탯 **힘**, 보조 **의지**. 열기 탈을 쓴 **물리 디펜더**(딜탱). 방어 불능 누적 특화.
※ 위키 각주5: 열기 부착/연소 효과 전무, 열기 피해만 줌 → `noArtsAttach`.

## 스킬
- **기본 돌진 검술**: 4단 물리. 강일 25 불균형.
- **배틀 진군** (게이지 100): 부채꼴 173%→390% 열기 + **넘어뜨리기**. 시전 중 피격 시 추가 불균형. 불균형 10(+10 피격).
- **연계 전선에서의 지원** (쿨 19~18초): **메인 피격 시**. 102%→230% 물리 + **넘어뜨리기 + 메인 치유**(300~675 + 의지×0.7~1.58).
- **궁극 다시 불타오르는 맹세** (게이지 100): 289%→650% 열기 광역 + **10초 팀 보호막 = 엠버 최대 HP의 18~25%**. 불균형 25.

## 재능
- **전진의 결의**: 배틀/연계 발동 중 50% 비호 + 스킬 안 끊김.
- **강철에는 강철로**: 피격 시 공격력 +9%(최대 3스택, 7초).

## 🎮 현재 구현
- skillMechanic `combo-strike`(이전 protective-arts 보호막봇 → 데미지+넘어뜨리기로 정정)
- operators: `physBreak: "build"`, `physAnomaly: "knockdown"`
- passive: `{ healOnCast: 12, damageResist: 0.12, selfPower: 0.06, noArtsAttach: true }`
  - `healOnCast 12` = 연계 치유(의지=장비 등급 비례) / `damageResist 0.12` = 전진의 결의 비호 / `selfPower 0.06` = 강철에는 강철로 / `noArtsAttach` = 열기 부착 억제
- 궁 팀 보호막 = `maxHp×0.25` (useUltimateOnState 커스텀). ULT_ENERGY 100
