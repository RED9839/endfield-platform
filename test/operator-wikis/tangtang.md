# 탕탕 (Tangtang / 청파채 대당가) — 냉기 / 권총 / 캐스터 ★6 (한정)

**육각형 냉기 캐스터**(유틸+서브딜). **즉발 냉기 부착(용오름)** + 용오름 개수 비례 **아츠 취약** + 와류(가속/감속) + **시간 정지 궁**(최대 4초 행동 불가) + 준수한 자체 딜. 에스텔라 외 유일 즉발 냉기 부착 → 라스트 라이트/이본 냉기 스트라이커 필수 보조. 메인딜로도 운용 가능. 주스탯 **민첩**, 보조 **힘**. 필라인(호랑이, 청파채/엔드필드). Lv90 HP 6000(한정 6성).

## 스킬 (설명문 원문)

### 기본 공격 「정신 똑바로 차려!」
최대 5단 냉기. 강력한 일격 18 불균형. **처형 400→900%**.

### 배틀 스킬 「우당탕탕 파도!」 (게이지 100)
> 근접 사격 냉기 + **용오름** 생성. 근처 모든 와류 소모 → 용오름 추가 생성, 개수만큼 게이지 반환.
> 용오름: 범위 냉기 부착 1스택 + 지속 냉기. 여러 개면 **아츠 취약** 추가(냉기 부착은 중복 X). 용오름 피해는 배틀 피해 취급.
- 사격 80→180% / 용오름 133→300% / 와류당 게이지 20 / 용오름 2개 아츠취약 6→10% / 3개 6→10%(별표기) / 불균형 10

### 연계 스킬 「야, 강물! 도와줘!」 (쿨 14~12초)
> 적이 **냉기 부착 / 아츠 폭발 피해**를 받았을 때. 물줄기 관통 냉기 + **와류 1개 생성**(최대 2개).
- 107→240% / 불균형 10 / 와류 지속 30초 / 쿨 14→12초

### 궁극기 「대당가께서 지켜보고 계신다!」 (게이지 90, 쿨 20초)
> 고대의 진 → 범위 적 일정 시간 **행동 불가**(시간 정지) + 지속 냉기. 변화 종료 시 거대 파도 대량 냉기. 메인 낙하 공격 시 변화 중단 + 강화 파도.
- 지속 142→320% / 진 4초 / 거대 파도 178→400% / 미리 소환 파도 311→700% / 불균형 15·20

## 재능 (설명문 원문)
- **의기투합**(1정): 와류 5m 내 아군 **가속 10%(1정)→20%(2정)**, 적 **감속 20%→40%**(이탈 후 3초 지속).
- **풍랑의 주재자**(2정): 진 내 메인 낙하 공격 시 변화 중단 + 용오름 1개(와류 소모로 추가), 배틀과 동일 효과 +40%(2정)→+60%(3정).
- 장비 조합 I/II/III(파랑/보라/노랑).

## 잠재능력 (명함 제외)
01 무한한 재보(연계 ×1.2·쿨 -2초·와류 소모 게이지 +5) / 02 강에서 태어난 소녀(민첩 +20·냉기 +10%) / 03 대당가의 기백(배틀 ×1.1·아츠취약 +5%) / 04 청파 수련법(궁 에너지 -15%) / 05 마안의 권능(궁 ×1.15·용오름 피해 +80%)

## 🎮 현재 구현 (DD류 — `app/game/dd/roster.ts` tangtang, Lv1 계수)
- OP_BASE: cls **caster**, hp 2689, attack 110, speed 74(민첩 최상위), ultCost **90**.
- **우당탕탕 파도!**(배틀, power 2.13=사격 80%+용오름 133%, `attach:cryo`, `target:row`) — 즉발 냉기 부착 + id훅: **와류 소모 → 용오름 1~3개**(개수 비례 아츠 취약 6%/8%) + 지속 냉기(dot 1.33×) + 와류당 게이지 +20. 검증: 와류0→용오름1(취약0), 와류2→용오름3(취약0.08·게이지+40) ✅
- **야, 강물! 도와줘!**(연계, power 1.07, cooldown 3, `requires` 냉기 부착/아츠 폭발) — 냉기 관통 + id훅: **와류 생성**(`procCount` 최대 2) + **의기투합**(아군 speedMod +10 / 적 -20). 검증: 와류 1·가속/감속 ✅
- **대당가께서…**(궁, power 3.11=미리 소환 파도, `target:all`, ultCost 90) — id훅: **시간 정지**(`stun` 타이머 → **상대 턴 1턴 뒤로**, 행동 불가) + 지속 냉기. 검증: stun true·행동 불가·1턴 뒤 복귀 ✅
- 엔진: 용오름/와류 카운터(`procCount` 재활용), 의기투합 가속/감속(`speedMod`), **시간 정지(`canAct`가 `timers.stun` 적 차단 → 1턴 스킵, 불균형 아님 = 받는 피해 보너스 없음)**. 신규 플래그 없이 id훅으로 구현.
  - ※ 불균형(`staggered`, +30% 받는 피해)과 구분: 시간 정지는 순수 행동 불가만(`stun` 타이머, startRound 감쇠 → 다음 라운드 복귀). 적이 이미 불균형이면 중복 부여 안 함.
- ⚠ 단순화: 용오름 다중 직접 피해(1개분+지속으로 근사), 풍랑의 주재자(낙하 공격), 와류 30초 지속/필드 위치, 잠재.

### 🔑 용오름/와류 순환 (`procCount`)
연계로 **와류** 적립(최대 2) → 배틀이 와류 소모해 **용오름** 1~3개 생성 → 용오름 개수↑ = 아츠 취약↑ + 게이지 반환↑. 즉발 냉기 부착은 에스텔라(쇄빙 한정)·자이히(비즉발)·알레쉬(궁 한정) 대비 유일하게 **원하는 타이밍**에 가능 → 냉기 스트라이커 코어. 시간 정지 궁은 staggered 재활용으로 극딜 윈도우 제공.
