# 포그라니치니크 (Pogranichnik) — 물리 / 한손검 / 뱅가드 ★6

주스탯 **의지**, 보조 **민첩**. **유일한 6성 뱅가드** — 갑옷 파괴(유일) + 스킬 게이지 수급 + 불균형 누적 + 조건부 버프. 전작 헬라그의 리컨비너. 자체 방불 부여는 궁뿐 → 팀 빌더 의존. 물리 파티 필수급.

---
## 스킬 (설명문 원문)

### 기본 공격 「전면 공세」
최대 5단 물리. 강일 18 불균형. 처형 400→900%.

### 배틀 스킬 「전선 분쇄」 (게이지 100)
> 전방 범위 두 번 베기 물리 + **갑옷 파괴**, 소모한 방어 불능 스택에 따라 **게이지 회복**(다중 명중 시 1회).
- 1단계 86→192% / 2단계 106→238% / 불균형 5+5 / **게이지 회복 5·10·20·30**(1~4스택, Lv1) → 5·15·25·35(Lv10)

### 연계 스킬 「보름달 참격」 (쿨 18~17초, 조건: 강타/갑옷파괴로 방불 소모됨)
> 소모한 최대 방불 스택 수만큼 베기(최대 3회) 물리 + 게이지 회복, 베기마다 피해·회복 증가. **방불 4스택 소모 시 3타 강화**.
- 1/2/3단계 42·54·66% → 95·122·149% / 3단계 강화 132→297% / 게이지 5·7·13 / 쿨 18→17초

### 궁극기 「방패병 부대, 전진」 (게이지 90)
> 방패병 4명 소환·진군, 경로 적 밀쳐내며 물리 + 30초간 **철의 서약 5포인트**. 적이 물리 이상/포그 연계 피해 받으면 철의 서약 1 소모 → 방패병 추가 소환(교란 물리 + 게이지). **마지막 1포인트 = 최후의 승부**(대량 물리 + 대량 게이지).
- 진군 133→300% / 교란 45→100% / 최후의 승부 200→450% / 불균형 10·15 / 게이지 7.5·30

## 재능 (설명문 원문)
- **생존의 깃발**: 자기 스킬로 게이지 80 회복마다 **사기 격양**(공격력 +4%/**+8%**, 아츠 강도 +4/**+8**, 20초, 최대 3스택).
- **전술 지도**: 임의 오퍼가 궁 후속 효과 발동 시 사기 격양 1스택(5/10초).

## 잠재능력 (명함 제외)
01 전선 소탕(배틀 2명+ 명중 시 게이지+15) / 02 행군(의지+20, 물리+10%) / 03 휘날리는 깃발(사기격양 60게이지/최대+2스택) / 04 탈로스 II의 방패(궁 에너지-15%) / 05 새로 단조한 검날(연계 쿨-2초·게이지×1.2)

## 🎮 현재 구현 (DD류 — `app/game/dd/roster.ts` pogranichnik, Lv1 계수)
- OP_BASE: cls **vanguard**(공식), hp 2689, attack 130(튜닝), speed **55**(민첩), ultCost 90.
- **전선 분쇄**(배틀, power 1.92, `armor-break`, `gaugeOnConsume:[5,10,20,30]`) — 갑옷 파괴 + 방불 N스택 소모 시 게이지 회복. 검증: 방불3 소모 → 게이지 +20 ✅
- **보름달 참격**(연계, power 1.62, `requires armor-break status`, cooldown 4, `gaugeGain:25`) — 갑옷파괴 후 사용 + 게이지 회복. 검증: 갑옷파괴 전 불가 → 후 가능 ✅
- **방패병 부대, 전진**(궁, power 1.33, `knockdown`, selfUlt, `grantsIronOath:5`) — 몹몰이 + 방불 부여 + **포그 본인에 철의 서약 5스택**.
- **철의 서약 추가타 체인**(정식 구현): 포그 궁 → 본인 5스택(6턴). 물리 이상/포그 연계가 적 타격 시 1 소모 → **교란**(포그 공격력 ×0.45 + 게이지 7.5), 마지막 1 = **최후의 승부**(×2.0 + 게이지 40). 검증: 5소모=교란4+최후1 ✅
- **생존의 깃발 사기 격양**(정식 구현): 포그가 자기 스킬로 게이지 **80 누적마다** atkBuff +0.08(최대 +0.24=3스택, 4턴). 검증: 전선분쇄×3(+30×3=90) → +0.08 ✅
- 엔진 신규: `gaugeGain`/`gaugeOnConsume`(게이지 수급) · `grantsIronOath`+`ironOath`(철의 서약, 쇄빙형 반응 트리거) · `gaugeRecovered`(사기 격양 누적, 칼날베기형 버프).
- ⚠ 미구현: 기본공격 처형, 잠재(명함 baseline 제외).
- ※ 카드판에선 "게이지 수급=템포 역효과"로 미적용했으나 **DD는 자원 경제라 정상 작동**(뱅가드 정체성 복원).
