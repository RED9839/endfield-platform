# 포그라니치니크 (Pogranichnik) — 물리 / 뱅가드 ★?

※ 위키 원문 배율표는 미보유(구현 기반 복원). 갑옷 파괴(관통) + 게이지 회복 + 철의 서약 방패병. physBreak **consume/armor-break**.

## 스킬
- **배틀 전선 분쇄** (전방 AoE): 2회 베기 물리 + **갑옷 파괴**(관통=받는 물리↑). 소모 방어 불능 스택 비례 게이지 회복(다중 명중 시 1회).
- **연계 보름달 참격** (단일, cond: 적이 강타/갑옷파괴로 방불 소모 후): 소모한 최대 방불 스택 비례 단계별 베기.
- **궁극 방패병 부대, 전진** (AoE): 방패병 4명 진군, 경로 밀쳐내며 물리 + **철의 서약 5포인트 생성**. 이후 적 물리 이상/포그 연계 시 1 소모(방패병 추가타+게이지).

## 재능
- **생존의 깃발**: 게이지 회복 시 사기 격양 공격 +8%.
- **처형 게이지 수급**(방불 소모 비례).

## 🎮 현재 구현
- passive: `{ breakEnergy: 1, selfPower: 0.08 }`
  - `breakEnergy 1` = 게이지 회복 / `selfPower 0.08` = 생존의 깃발(상시 화력)
- 궁 = ironOath 5(useUltimateOnState). physBreak consume/armor-break(관통). ULT_ENERGY 90
- ⚠ 위키 '방불 소모 비례 게이지 수급'(`gaugeOnArmorBreak`)은 **카드-템포 모델에서 에너지=템포라 역효과**(sim 44%로 하락) → 미적용. 인프라는 비활성 보존
