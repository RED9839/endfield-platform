# 아비웨나 (Avywenna / 본명 람포) — 전기 / 장병기 / 스트라이커 ★5

**전기팟 핵심 딜러 · 썬더랜스(투창) 설치-회수 메커니즘**. 연계/궁으로 **썬더랜스를 필드에 설치(30초 잔존)** → 배틀로 **전부 회수하며 경로 적에 투창 수 비례 중복 폭딜**. 투창이 많을수록 폭딜↑. **전기 부착/감전을 소모하지 않음**(연계 조건일 뿐) → 타 전기 스트라이커와 더블 딜러 가능. 5성이라 전기팟 쌀먹 구원자. 주스탯 **의지**, 보조 **민첩**. 카우투스(상업연합회 전달자).

---
## 스킬 (설명문 원문)

### 기본 공격 「썬더랜스 · 신속 공격」
최대 5단 **물리**. 강력한 일격 17 불균형. **처형 400→900%**.

### 배틀 스킬 「썬더랜스 · 가로채기」 (게이지 100)
> 점프 창 회오리 소량 전기 + **모든 썬더랜스/강력한 썬더랜스 회수**. 회수 경로 적 명중 시 전기. 강력 썬더랜스는 큰 전기 + **전기 부착**.
- 본체 67→150% / 썬더랜스 75→168% / 강력 썬더랜스 192→432%(+전기 부착)

### 연계 스킬 「썬더랜스 · 번개 타격」 (쿨 13~12초)
> 메인이 **전기 부착/감전 적**에 강력한 일격 후. 점프 → **30초 존재하는 썬더랜스 3개** 던져 전방 범위 전기.
- 169→380% / 쿨 13→12초

### 궁극기 「썬더랜스 · 결전의 떨림」 (게이지 100, 쿨 10초)
> **30초 존재하는 강력한 썬더랜스 1개** 투척 + 주변 대량 전기.
- 422→950% / 불균형 15

## 재능 (설명문 원문)
- **고효율 배송**(1정): 투척/회수한 썬더랜스·강력 썬더랜스 명중 시 **궁 에너지 +3(1정)→+4(2정)**.
- **완곡한 수단**(2정): 궁 명중 시 **전기 취약 6%(2정)→10%(3정)**(10초).
- 장비 조합 I/II/III(파랑/보라/노랑).

## 잠재능력 (명함 제외)
01 더블 베팅(고효율 배송 +2) / 02 눈앞에 닥친 위험(썬더랜스 지속 +20초) / 03 끝없는 저항(의지 +15·전기 +8%) / 04 숙련자의 여유(궁 에너지 -15%) / 05 회유와 위압(회수 투창이 전기 취약 적 명중 시 ×1.15)

## 🎮 현재 구현 (DD류 — `app/game/dd/roster.ts` avywenna, Lv1 계수)
- OP_BASE: cls **striker**, hp 2689, attack 110, speed 62, ultCost **100**. 평타 물리·스킬 전기. 주스탯 의지.
- **썬더랜스·가로채기**(배틀, power 0.67, `lanceRecover:true`) — 설치 투창 **전부 회수** → 본체 + (일반×0.75 + 강력×1.92) 중복 전기타 + 강력 투창 전기 부착 + 고효율 배송 궁충. 검증: 6일반+1강력 회수 → 7.09×(709%) 폭딜·재부착·리셋 ✅
- **썬더랜스·번개 타격**(연계, power 1.69, `target:row`, cooldown 3, `requires` 전기 부착/감전) — 전기 + id훅: **일반 썬더랜스 3개 설치**(`procCount`) + 궁충 +12. **부착 미소모**. 검증: 2회 → 일반 6·전기 부착 유지 ✅
- **썬더랜스·결전의 떨림**(궁, power 4.22, `target:all`, ultCost 100) — 광역 전기 + id훅: **강력 썬더랜스 1개 설치**(`gaugeRecovered`) + `apply` **완곡한 수단**(전기 취약 0.10). 검증: 강력 1·전기 취약 0.10 ✅
- 엔진 신규: **`lanceRecover`**(투창 수 비례 중복타 + 강력 부착 + 고효율 궁충 + 회수 리셋). 투창 카운터는 `procCount`(일반)·`gaugeRecovered`(강력) 재활용(아비웨나 전용 — 타 유닛과 무충돌).
- ⚠ 단순화: 투창 30초 지속(영구 보유 → 다음 배틀 회수, 만료 미모델), 회수 풀히트 포지셔닝(단일 극딜로 근사), 평타 물리 vs 스킬 전기, 잠재.

### 🔑 설치-회수 사이클 (부착 미소모 = 더블 스트라이커)
연계(일반 3개) ×N + 궁(강력 1개) 설치 → **배틀 가로채기로 전부 회수 = 투창 수 비례 폭딜**. 라스트 라이트/이본(부착 소모형)과 달리 **전기 부착/감전을 소모하지 않아** 펠리카 감전·아크라이트 소모와 충돌 없이 공존 → 전기팟 더블 스트라이커 가능. 질베르타(짧은 극딜 윈도우·아츠 취약)와 궁합 최상.
