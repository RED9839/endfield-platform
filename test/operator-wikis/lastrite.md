# 라스트 라이트 (Last Rite / 본명 아델하이드) — 냉기 / 양손검 / 스트라이커 ★6 (한정)

**첫 스트라이커 · 대 보스 단일 누킹 특화**. 냉기 부착 3+스택을 연계로 소모 → 압도적 단일 누킹 + **저체온증**(냉기 취약) + **저온 취성**(궁 취약 1.5배). 궁 게이지를 **자신의 배틀/연계로만** 충전(240). 느린 양손검·다수전 약점(탕탕으로 보완). 살카즈(나흐체러르, 세쉬카). 주스탯 **힘**, 보조 **의지**.

---
## 스킬 (설명문 원문)

### 기본 공격 「혹한의 춤」
최대 4단 냉기. 강력한 일격 25 불균형(높음). **처형 400→900%**.

### 배틀 스킬 「세쉬카의 비전」 (게이지 100, **30 반환**)
> 15초간 메인 무기에 저온 주입 + 게이지 30 반환. 메인 강력한 일격 1회 → 라라 환영 추격 → 냉기 + **냉기 부착** + **궁 에너지 16**. 라라가 메인이면 일반 공격 안 끊김.
- 환영 추격 142→320%

### 연계 스킬 「겨울 포식자」 (쿨 9~8초)
> **냉기 부착 3스택 이상**일 때. 얼음송곳 응결 후 분쇄. 냉기 부착 **전부 소모** → 스택 수치 비례 냉기 피해 + 소모 스택 비례 궁 에너지. 얼음송곳 강제 정지.
- 얼음송곳 71→160% / 베기 71→160% / 스택당 107→240% / 기초 궁 40 + 스택당 15 / 불균형 15

### 궁극기 「마지막 인사」 (게이지 **240**, 쿨 20초)
> 서리 갑옷 3회 베기, **시전 중 피해 면역**. 대량 냉기. **배틀/연계로만 궁 에너지 충전 가능**.
- 1·2단 178→400% / 3단 356→800% / 불균형 5+5+10

## 재능 (설명문 원문)
- **저체온증**(1정): 임의의 아츠 부착 소모 시 **소모 스택 ×2%(1정)→×4%(2정) 냉기 취약**(15초, 비중첩).
- **저온 취성**(2정): 궁이 냉기 취약 적에 피해 시 **냉기 취약 ×1.2(2정)→×1.5(3정) 간주**(아츠 취약 상위 카테고리 포함).
- 장비 조합 I/II/III(파랑/보라/노랑).

## 잠재능력 (명함 제외)
01 묘지기의 선물(저온 주입 강일 +20%·불균형 +5) / 02 영점 무장(힘 +20·냉기 +10%) / 03 혹한의 지배자(연계·궁 ×1.15) / 04 진심 어린 작별(궁 에너지 -15%) / 05 다시 밀려오는 한기(배틀 게이지 +5·환영 ×1.2)

## 🎮 현재 구현 (DD류 — `app/game/dd/roster.ts` lastrite, Lv1 계수)
- OP_BASE: cls **striker**(첫 스트라이커!), hp 2689, attack 110, speed 48(느림), ultCost **240**(자기 충전). 주스탯 힘.
- **세쉬카의 비전**(배틀, power 1.42, `attach:cryo`, gaugeRefund 30) — 냉기 부착(환영 추격) + id훅 **궁 에너지 +16**(자기 충전). 검증: 냉기 부착 1·라라 궁 22.5 ✅
- **겨울 포식자**(연계, power 1.42, cooldown 2, `cryoNuke:1.07`, `requires` 냉기 3스택+) — 냉기 **전부 소모** → 스택당 누킹 + **저체온증**(스택×4% 냉기 취약) + **강제 정지**(stun) + 궁 +50. 검증: 냉기 4스택 소모·냉기취약 0.16·강제 정지 ✅
- **마지막 인사**(궁, power 7.12, `target:single-front`, ultCost 240) — 단일 3연 누킹 + **저온 취성**(`vulnFor ×1.5`). 검증: 피해 971 = raw×(1+취약 0.16×1.5) ✅
- 엔진 신규: **`cryoNuke`**(냉기 소모 스택 비례 누킹+저체온증+강제 정지), **저온 취성**(라라 궁 한정 취약 ×1.5), **궁 자기 충전 락**(`lastrite`는 타 아군 배틀의 ULT_BATTLE 무효 — 배틀/연계 자가 충전만). 검증: 타 아군 배틀 → 라라 궁 0 ✅
- ⚠ 단순화: 저온 주입 "메인 강일 트리거"(배틀 즉시 환영으로 근사), 궁 시전 중 피해 면역(즉발), 연계 강제 정지 = stun(탕탕과 공유), 잠재.

### 🔑 냉기 부착 누킹 사이클 + 자기 충전 궁
배틀(냉기 부착 +1) ×N + 파트너(탕탕 즉발/자이히 강일/스노우샤인 반격) → 냉기 3~4스택 → 연계 겨울 포식자로 **전부 소모 = 스택 비례 폭딜 + 저체온증 냉기 취약** → 궁 마지막 인사(저온 취성으로 취약 1.5배) 마무리. 궁 240은 **라라 본인 배틀(+22.5)/연계(+60)로만** 충전 → ~3사이클. 탕탕(즉발 냉기 부착·다수전)·자이히(강일 냉기 부착·아츠/냉기 증폭)가 코어 파츠.
