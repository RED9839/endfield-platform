# 카치르 (Catcher / 본명 루이 판 카치르) — 물리 / 양손검 / 디펜더 ★4

**정석 패링 탱커**(오픈 기준 가장 정석적 디펜더). 어떤 공격이든 1회 막는 배틀(비호+슈퍼아머) + 반격 **방어 불능 부여**(물리 패링) + 보호막 + 허약/넘어뜨리기 궁. 주스탯 **힘**, 보조 **의지**. 페로(엔드필드 Z7). 엠버 하위 호환·회피 메타에서 취직처 좁지만 **2디펜더 극한 안정성**·소울류 1인 플레이.

---
## 스킬 (설명문 원문)

### 기본 공격 「기초 전술」
최대 4단 물리. 강력한 일격 22 불균형. **처형 400→900%**.

### 배틀 스킬 「강력한 저지」 (게이지 100, **30 반환**)
> 방패 들어 자신+주변 오퍼레이터에게 **90% 비호** + 게이지 30 반환. 방패 든 동안 **공격받으면 반격** → 물리 + **방어 불능 1스택**.
- 178→400% / 불균형 20 / 비호 90%(메인 슈퍼아머)

### 연계 스킬 「실시간 억제」 (쿨 35~33초)
> **적이 차지 시작 / 메인 컨트롤 오퍼레이터 생명력 40% 이하**일 때. 주먹 내려꽂아 물리 + 자신+다른 아군(메인 우선) **10초 보호**. **방어력**이 보호 수치 증가.
- 1단 25→55% / 2단 100→225% / 기초 보호 360→810 / 방어력당 2.25→5.06 / 불균형 10 / 쿨 35→33초

### 궁극기 「교과서적인 맹공」 (게이지 80, 쿨 15초)
> 2단 베기 물리 + 8초 **허약**. 마지막 내리찍어 범위 모든 적 대량 물리 + **넘어뜨리기**.
- 1단 89→200% / 2단 120→270% / 3단 178→400% / 허약 20→30% / 불균형 5+5+10
- **전장을 꿰뚫는 통찰**(재능): 마지막 공격 충격파 2(2정)/3(3정)회, 각 공격력 30/45% 물리

## 재능 (설명문 원문)
- **강인한 방어선**(1정): 의지 10마다 방어력 +1.0(1정)→+1.2(2정).
- **전장을 꿰뚫는 통찰**(2정): 궁 마지막 공격 충격파 2(2정)/3(3정)회 × 공격력 30/45% 물리.
- 장비 조합 I/II/III(파랑/보라/노랑).

## 잠재능력 (명함 제외)
01 다중 전투준비(배틀·궁 명중 후 [300+방어력×5] 물리) / 02 추가 특훈(방어력+20·의지+10) / 03 불굴의 심장(연계 보호 +5초) / 04 고난의 대가(궁 에너지 -10%) / 05 후회 없는 선택(보호 중 배틀 명중 시 게이지 +10)

## 🎮 현재 구현 (DD류 — `app/game/dd/roster.ts` catcher, Lv1 계수)
- OP_BASE: cls **defender**, hp 2689, attack 110, speed **42**, ultCost **80**.
- **강력한 저지**(배틀, `target:self`, power 0, gaugeRefund 30) — id훅: 자신+주변 **90% 비호** + **반격 태세**(`timers.guard`). 검증: 비호0.9·게이지100→30·태세 ✅
- **실시간 억제**(연계, power 1.25, cooldown 7, `requires` 아군 HP≤40%) — 물리 + id훅: 자신+아군 **보호막**(`360+gearGrade×2.25`=495, **강인한 방어선** 방어력 비례). 검증: 보호막 495 ✅
- **교과서적인 맹공**(궁, power **5.22**=387%+충격파 135%, `anomaly:knockdown`, apply 허약 0.2) — 다단 물리 + 허약 + 광역 넘어뜨리기. 검증: 허약 0.80·방불 2(반격1+넘어뜨리기1) ✅
- 엔진: **패링 반격 분기 확장** — 스노우샤인(냉기 부착)과 같은 `timers.guard` 윈도우, 카치르는 **방어 불능 1스택**(`physBreak+1`) 부여로 분기. 충격파(전장을 꿰뚫는 통찰)·강인한 방어선은 power/`gearGrade` 베이크.
- ⚠ 단순화: 적 차지 트리거(HP 40%만), 보호막 방어력은 `gearGrade` 치환, 슈퍼아머·잠재(01 추가타·03 보호 연장).
