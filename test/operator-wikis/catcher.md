# 카치르 (Catcher) — 물리 / 양손검 / 디펜더 ★4

주스탯 **힘**, 보조 **의지**. 패링 디펜더. 물리라 반격이 방어 불능 스택.

## 스킬
- **기본 기초 전술**: 4단 물리. 강일 22 불균형.
- **배틀 강력한 저지** (게이지 100, **30 반환**): 자신+주변 **90% 비호**. 막는 동안 피격 시 반격 → 178%→400% 물리 + **방어 불능 1스택**. 불균형 20.
- **연계 실시간 억제** (쿨 35~33초): **적 차지 or 메인 HP 40% 이하**. 물리(25%/100%→55%/225%) + 보호(360~810 + 방어력×2.25~5). 불균형 10.
- **궁극 교과서적인 맹공** (게이지 80): 2단 베기 물리 + **허약 20~30%(8초)** + 마지막 광역 물리 **넘어뜨리기**. 불균형 5/5/10.

## 재능
- **강인한 방어선**: 의지 10마다 방어력 +1.2.
- **전장을 꿰뚫는 통찰**: 궁 마지막 공격이 충격파 3회(각 공격력 45% 물리).

## 🎮 현재 구현
- skillMechanic `protective-arts` (**배틀만** 보호막; 연계·궁 정상 카드)
- operators: `physBreak: "build"`, `physAnomaly: "knockdown"`(궁 넘어뜨리기)
- passive: `{ shieldOnCast: 14, reflectAttach: true, damageResist: 0.12, breakEnergy: 1 }`
  - `reflectAttach`(물리 → 반격 방어 불능 1스택 + 게이지) / `damageResist 0.12` = 90% 비호 = 강인한 방어선 / `shieldOnCast 14` = 실시간 억제 보호 / `breakEnergy 1` = 게이지 반환
- ULT_ENERGY 80. (위키상 엠버 하위호환)
