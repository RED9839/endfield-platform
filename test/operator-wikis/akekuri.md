# 아케쿠리 (Akekuri) — 열기 / 한손검 / 뱅가드 ★4

주스탯 **민첩**, 보조 **지능**. **속성 무관 범용 게이지 수급 뱅가드**(탈4성). 불균형 조건 연계 + 무딜 궁(게이지 대량 회복) + 연타(몰입의 시간). "엔드필드의 머틀". 열기 부착도 보유해 열기팟 서폿.

---
## 스킬 (설명문 원문)

### 기본 공격 「진취의 검날」
최대 4단 물리. 강일 17 불균형. 처형 400→900%.

### 배틀 스킬 「열정 분출」 (게이지 100)
> 전방 검 열기 + **열기 부착**.
- 142→320% / 불균형 10

### 연계 스킬 「섬광 돌진」 (쿨 10~9초, 조건: **불균형 상태/불균형 지점 적**)
> 2연속 찌르기 물리, 각 게이지 7.5 회복. **딜타임 직전 확정 발동** → 가장 필요할 때 게이지 공급.
- 단계별 80→180%(×2) / 게이지 7.5×2=15 / 불균형 5 / 쿨 10→9초

### 궁극기 「소대, 집합!」 (게이지 120)
> 지속 시전, 신호탄 3발. 발사마다 게이지 회복. **딜 없음 — 순수 게이지 회복 궁.**
- 총 게이지 회복 58→80

## 재능 (설명문 원문)
- **승리의 함성**: 지능 10당 섬광 돌진 게이지 회복 +1.0%/**+1.5%**, 최대 +50%/**+75%**.
- **몰입의 시간**: 궁 소대 집합 지속 중 **연타** 획득(다음 배틀/궁 강화).

## 🎮 현재 구현 (DD류 — `app/game/dd/roster.ts` akekuri, Lv1 계수)
- OP_BASE: cls **vanguard**, hp 2689, attack 110(튜닝), speed **70**(민첩), ultCost **120**.
