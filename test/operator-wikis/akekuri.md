# 아케쿠리 (Akekuri) — 열기 / 한손검 / 뱅가드 ★4

주스탯 **민첩**, 보조 **지능**. 게이지 회복 배터리 + 열기 부착 + 궁 중 연타.

## 스킬
- **기본 진취의 검날**: 4단 물리. 강일 17 불균형.
- **배틀 열정 분출** (게이지 100): 전방 142%→320% 열기 + 열기 부착. 불균형 10.
- **연계 섬광 돌진** (쿨 10~9초): **불균형 상태/지점 적 있을 때**. 2연속 찌르기 80%→180%(각), 게이지 7.5 회복(타). 불균형 5.
- **궁극 소대, 집합!** (게이지 120): 지속 시전, 신호탄 3발. 총 게이지 회복 58~80. (딜 미미, 게이지 회복 궁)

## 재능
- **승리의 함성**: 지능 10마다 연계 게이지 회복 +1.5%, 최대 +75%.
- **몰입의 시간**: 궁극 지속 중 연타 획득.

## 🎮 현재 구현
- passive: `{ breakEnergy: 1, grantMultiHit: true, linkStatGauge: 2 }`
  - `breakEnergy 1` = 불균형 돌파 시 에너지
  - `grantMultiHit` = 몰입의 시간(궁 중 연타)
  - `linkStatGauge 2` = 승리의 함성(연계 추가 에너지 +지능=장비 등급 비례). 조건부(불균형 적)라 0.05 같은 상시 teamAmp와 달리 sim 안전
- ULT_KIND **energy**(집결 신호탄 → 전투 에너지 회복), ULT_ENERGY 120
