# 울프가드 (Wulfgard) — 열기 / 권총 / 캐스터 ★5

주스탯 **힘**, 보조 **민첩**. 열기 부착 + 연소 + 아츠 이상 소모 추가타. 연소 파티 핵심 딜러(로시 오빠).

## 스킬
- **기본 다중 연사**: 4단 열기. 강일 18 불균형.
- **배틀 탄흔의 열기** (게이지 100): 연속 사격 102%→230% 열기, 마지막 **열기 부착**. **목표가 연소/감전이면 부착 대신 그 상태 소모 + 추가 1회 사격(378%→850% 대량 열기)**. 불균형 5+5.
- **연계 폭렬 수류탄·β형** (쿨 20~19초): **아츠 부착 적 있을 때**. 범위 60%→135% 열기 + **열기 부착**. 불균형 10.
- **궁극 늑대의 분노** (게이지 90): 5회 32%→72% 열기 + **강제 연소**. 불균형 15.

## 재능
- **불타는 송곳니**: 연소 부여 시마다 불타는 송곳니(10초) → 자기 열기 피해 +20~30%. (잠재3 사냥 시간: 배틀 추가효과 시 팀에 50% 분배)
- **절제의 원칙**: 배틀로 아츠 이상 소모 시 게이지 +5~10 반환.

## 🎮 현재 구현
- skillMechanic `combo-strike`(열기 부착)
- passive: `{ selfPower: 0.18, breakEnergy: 1, ultForceCombustion: true, consumeArtsBonus: 1.4 }`
  - `selfPower 0.18` = 불타는 송곳니(상시 화력으로 근사) / `breakEnergy 1` + ULT_ARTS_CONSUME 15 = 절제의 원칙
  - `ultForceCombustion` = 늑대의 분노 강제 연소(궁 적중 적, useUltimateOnState 후처리)
  - `consumeArtsBonus 1.4` = 탄흔의 열기 연소/감전 소모 추가타(카드 위력 ×1.4, 소모 시 열기 부착 생략). 위키 배율 3.7x는 카드 경제상 과해서 1.4로 보정
- ULT_ENERGY 90. 사냥 시간(팀 분배)은 미모델
