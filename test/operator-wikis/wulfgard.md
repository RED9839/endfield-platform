# 울프가드 (Wulfgard / 카텔로 울프가드 루피노) — 열기 / 권총 / 캐스터 ★5

**연소 파티 핵심 딜러**(로시의 오빠, 울프팀 랜드브레이커). 열기 부착 + **연소/감전 소모 대량 추가타(378%+)** + 강제 연소 궁 + **불타는 송곳니**(연소 부여 시 열기 증폭). 레바테인/로시 열기팟 핵심. 위기협약 재점화(아츠 이상 +100%)에서 5성 이례적 고득점 딜러. 주스탯 **힘**, 보조 **민첩**. 루포(엔드필드 공업).

---
## 스킬 (설명문 원문)

### 기본 공격 「다중 연사」
최대 4단 열기. 강력한 일격 18 불균형. **처형 400→900%**.

### 배틀 스킬 「탄흔의 열기」 (게이지 100)
> 연속 사격 소량 열기 + 마지막 **열기 부착**. 목표가 **연소/감전 상태면 열기 부착 대신 그 상태 소모, 추가 1회 사격 대량 열기**.
- 102→230% / 추가 378→850% / 불균형 5 + 추가 5

### 연계 스킬 「폭렬 수류탄 · β형」 (쿨 20~19초)
> **아츠 부착 상태 부여한 적이 있을 때**. 폭렬 수류탄 → 범위 열기 + **열기 부착**.
- 60→135% / 불균형 10 / 쿨 20→19초

### 궁극기 「늑대의 분노」 (게이지 90, 쿨 10초)
> 빠른 사격 5회 열기 + **강제 연소**.
- 각 32→72%(×5=160→360%) / 불균형 15

## 재능 (설명문 원문)
- **불타는 송곳니**(1정): 연소 부여마다 불타는 송곳니(10초, 비중첩) → 자기 열기 피해 +20%(1정)→+30%(2정).
- **절제의 원칙**(2정): 배틀로 아츠 이상 소모 시 게이지 +5(2정)→+10(3정) 반환.
- 잠재3 사냥 시간: 배틀 추가효과 시 불타는 송곳니 팀 50% 분배(명함 제외).

## 잠재능력 (명함 제외)
01 떠돌이 늑대(힘·민첩 +15) / 02 총기 개조(절제의 원칙 +10) / 03 사냥 시간(불송 초기화 + 팀 50% 분배) / 04 울프팀의 의지(궁 에너지 -15%) / 05 타고난 포식자(궁 후 연계 쿨 초기화)

## 🎮 현재 구현 (DD류 — `app/game/dd/roster.ts` wulfgard, Lv1 계수)
- OP_BASE: cls **caster**, hp 2689, attack 110, speed 70, ultCost **90**. 주스탯 힘.
- **탄흔의 열기**(배틀, power 1.02, `attach:heat`, `burnShockConsume:3.78`) — 연소/감전 적이면 **부착 생략 + 소모 → 추가 378% 열기 + 게이지 +10**(절제의 원칙), 아니면 열기 부착. 검증: 부착→연소 소모(combustion 해제·게이지 순 -90) ✅
- **폭렬 수류탄·β형**(연계, power 0.6, `target:row`, `attach:heat`, cooldown 4, `requires` 아츠 부착) — 범위 열기 부착(부착 셋업). 검증: 부착 적 게이트 ✅
- **늑대의 분노**(궁, power 1.6, `target:all`, `forceBurn:true`, ultCost 90) — 광역 열기 + **강제 연소**(부착 무관 dot+combustion) + **불타는 송곳니**(self amp.heat 0.30). 검증: 강제 연소·열기 증폭 0.30 ✅
- 엔진 신규: **`burnShockConsume`**(연소/감전 소모 추가타, 소모 시 부착 생략 + 절제의 원칙 게이지), **`forceBurn`**(강제 연소), **불타는 송곳니**(울프가드가 연소 부여 시 self 열기 +30%, 위키 배율 그대로). `combustion` 마커(질베르타와 공유) 활용.
- ⚠ 단순화: 사냥 시간(팀 분배), 권총 다단(단일 배율), 잠재.

### 🔑 연소/감전 소모 사이클 (위키 배율 충실)
DD류는 카드 게이지 경제가 없어 **위키 추가타 배율 378%를 그대로 반영**(카드 모델 시절 1.4배 보정과 대비). 부착(배틀/연계) → 다른 속성 부착으로 연소/감전 이상 → 탄흔의 열기로 소모 → 대량 추가타 + 게이지 반환. 불타는 송곳니가 연소마다 self 화력 +30% 누적. 레바테인 연계(연소 조건)·로시 하이브리드와 직결.
