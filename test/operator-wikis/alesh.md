주스탯 **힘**, 보조 **지능**. **강제 동결**(냉기 단독 동결!) + 게이지 수급 + 아츠 이상/결정 소모 연계 + 린수 확률. **에스텔라 쇄빙 파티 핵심**(동결 공급). 낮은 운용 부담(연계 위주 무게이지 운용 가능).

- 200→450% / 게이지 회복 10·20·30·40(→Lv10 15·25·35·45) / 불균형 10

- 133→300% / 강화 213→480% / 게이지 10~13 / 쿨 9→8초

- 436→980% / 게이지 20→25 / 처치 +12→15 / 불균형 20

## 🎮 현재 구현 (DD류 — `app/game/dd/roster.ts` alesh, Lv1 계수)
