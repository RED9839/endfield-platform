# 알레쉬 (Alesh) — 냉기 / 한손검 / 뱅가드 ★5

주스탯 **힘**, 보조 **지능**. 강제 동결 + 게이지 회복 + 아츠 이상 소모 연계.

## 스킬
- **기본 캐스팅의 기본**: 5단 물리. 강일 17 불균형.
- **배틀 비정규 루어** (게이지 100): 얼음 내리침 200%→450% 물리. **냉기 부착 적 명중 시 냉기 전부 소모 + 강제 동결**, 소모 스택 비례 게이지 회복(10/20/30/40 → 15/25/35/45). 불균형 10.
- **연계 얼음 낚시 기술** (쿨 9~8초): 근처 적 **아츠 이상 or 오리지늄 결정 소모 시** 발동. 133%→300%(강화 213%→480%) 물리 + 게이지 회복 10~13. 10% 확률 진귀한 린수 → 피해 대폭↑ + 게이지 +10.
- **궁극 월척이다!** (게이지 100): 전방 모든 적 436%→980% 냉기 광역 + 냉기 부착 + 게이지 회복 20~25. 처치 시 추가 게이지(최대 100). 불균형 20.

## 재능
- **급속 냉동 보존 기술**: 동결/오리지늄 결정 부착 후 궁에너지 +4, 자기 동결 발동 시 +8 추가(3초 1회).
- **낚시의 달인**: 지능 10마다 린수 확률 +0.5%, 최대 +30%.

## 🎮 현재 구현
- passive: `{ ultOnFreeze: 8, forceFreezeOnCryo: true, linkStatDamage: 0.12 }`
  - `ultOnFreeze 8` = 급속 냉동(동결 부여 시 궁충)
  - `forceFreezeOnCryo` = 비정규 루어 강제 동결(냉기 부착 적 → 동결, 단일 원소로도)
  - `linkStatDamage 0.12` = 낚시의 달인(연계 피해 +지능=장비 등급 비례, 린수 평균값)
- ULT_ENERGY 100, ULT_ARTS_CONSUME 20(게이지 회복형)
