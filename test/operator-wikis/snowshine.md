# 스노우샤인 (Snowshine) — 냉기 / 양손검 / 디펜더 ★5

주스탯 **힘**, 보조 **의지**. 패링(방패 막기) 디펜더. 저비용 냉기 부착 + 강제 동결.

## 스킬
- **기본 저체온 강타**: 3단 물리. 강일 23 불균형.
- **배틀 포화성 방어** (게이지 100, **30 반환**): 자신+주변 **90% 비호**. 막는 동안 피격 시 반격 → 200%→450% 냉기 + **냉기 부착**. 불균형 20.
- **연계 극지 구조** (쿨 25~23초): **메인 HP 60% 이하**. 대량 치유(초기 96~216 + 의지×0.22~0.5) + 3초 지속 치유.
- **궁극 살얼음 추위** (게이지 80): 200%→450% 냉기 광역 + 빙설 지대(0.5초마다 지속 냉기) + **일정 시간 못 벗어나면 강제 동결**. 불균형 15.

## 재능
- **극지 생존**: HP 55% 이하 목표 치유 +25%.
- **구조 전문가**: 배틀 반격 시 궁에너지 +10.

## 🎮 현재 구현
- skillMechanic `protective-arts` (**배틀만** 보호막=비호; 연계 치유·궁 딜은 정상 카드)
- passive: `{ healOnCast: 12, breakEnergy: 1, damageResist: 0.12, reflectAttach: true, ultForceFreeze: true }`
  - `reflectAttach` = 포화성 방어(보호막 + 반격 냉기 부착 + 게이지 반환). 냉기 오퍼라 부착, 물리 오퍼(카치르)면 방어 불능
  - `damageResist 0.12` = 90% 비호 / `healOnCast 12` = 극지 구조 치유 / `breakEnergy 1` = 구조 전문가
  - `ultForceFreeze` = 살얼음 추위 강제 동결(궁 적중 적, 냉기 부착 미소모)
- ULT_ENERGY 80
