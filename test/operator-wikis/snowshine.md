# 스노우샤인 (Snowshine / 전작 오로라의 리컨비너) — 냉기 / 양손검 / 디펜더 ★5

**글로벌 사전 예약 배포** 디펜더. **패링(방패 막기) + 저비용 냉기 부착(반격)** + 비호 + 치유 + 궁 강제 동결(부착 미소모, 쇄빙 보조). 주스탯 **힘**, 보조 **의지**. 우르수스(로도스 아일랜드). 낮은 육성 비용·높은 파티 안정성이지만 딜 보조 빈약·반격 의존도 단점. 그림자 이정표(고피해 상시) 만능.

---
## 스킬 (설명문 원문)

### 기본 공격 「저체온 강타」
최대 3단 물리. 강력한 일격 23 불균형. **처형 400→900%**.

### 배틀 스킬 「포화성 방어」 (게이지 100, **30 반환**)
> 방패 들어 자신+주변 오퍼레이터에게 **90% 비호** + 게이지 30 반환. 방패 든 동안 **공격받으면 반격** → 적에게 냉기 + **냉기 부착**.
- 200→450% / 불균형 20 / 비호 90%(메인은 슈퍼아머)

### 연계 스킬 「극지 구조」 (쿨 25~23초)
> **메인 컨트롤 오퍼레이터가 공격받아 생명력 60% 이하**일 때. 설원 구조 조수 → 명중 시 주변 오퍼레이터 **대량 치유** + 3초간 0.5초마다 지속 치유. **의지** 치유량 증가.
- 초기 치유 96→216 / 의지당 0.22→0.5 / 지속 치유 24→54 / 의지당 0.06→0.13 / 쿨 25→23초

### 궁극기 「살얼음 추위」 (게이지 80, 쿨 20초)
> 점프 후 방패 내리찍어 범위 적 **대량 냉기** + 5초간 빙설 지대(0.5초마다 냉기). **일정 시간 벗어나지 못하면 강제 동결**.
- 폭발 200→450% / 불균형 15(Lv10+ 20) / 지속 29→65%

## 재능 (설명문 원문)
- **극지 생존**(1정): HP 45%(1정)→55%(2정) 이하 목표 **치유 +15→25%**.
- **구조 전문가**(2정): 포화성 방어 **반격 성공 시 궁 에너지 +6(2정)→+10(3정)**.
- 장비 조합 I/II/III(파랑/보라/노랑).

## 잠재능력 (명함 제외)
01 저체온 대피소(비호 중 아군 아츠 부착 면역) / 02 눈보라 지대(궁 범위 +20%) / 03 극지 생존 가이드(궁 동결 +2초) / 04 얼어붙은 대지의 방패(방어력·의지 +20) / 05 한파 대응 전문가(반격 성공 시 게이지 +10)

## 🎮 현재 구현 (DD류 — `app/game/dd/roster.ts` snowshine, Lv1 계수)
- OP_BASE: cls **defender**, hp 2689, attack 110, speed **44**, ultCost **80**(저비용 궁).
- **포화성 방어**(배틀, `target:self`, power 0, gaugeRefund 30) — id훅: 자신+주변 **90% 비호**(applyBuff protection 0.9, 1턴) + **반격 태세**(`timers.guard`). 검증: 비호0.9·게이지100→30·태세 ✅
- **극지 구조**(연계, `target:self`, power 0, cooldown 5, `requires` 아군 HP≤60%) — id훅: 아군 대량 치유(`96+gearGrade×0.22`=109) × **극지 생존**(HP≤55% ×1.25). 검증: 에스텔라 +136 ✅
- **살얼음 추위**(궁, power 2.0, `element:cryo`, `freezeZone:1`, ultCost 80) — 광역 냉기 + **빙설 지대 강제 동결**(부착 무관) + 지속 냉기. 검증: 동결 1(부착 미소모)→띄우기 쇄빙 소모 ✅
- 엔진 신규: **반격(패링)** — 아군 피격 시 `timers.guard` 활성 스노우샤인 → 공격자에 `applyAttach cryo`(저비용 냉기 부착) + **구조 전문가** 궁 +10. **`freezeZone`** 플래그(부착 무관 직접 동결 + 지속 냉기) — 알레쉬 `forceFreeze`(부착 소모)와 구별, 쇄빙 트리거 호환.
- ⚠ 단순화: 반격 200% 냉기 피해(부착만 모델), 지속 치유(초기 치유에 통합), 의지→`gearGrade` 치환, 슈퍼아머·잠재(01 부착 면역·03 동결 연장).

### 🔑 반격(패링) vs 빙설 지대
- **반격**: `timers.guard`(라운드 한정, startRound 감쇠). 적이 아군 유효타 → 스노우샤인 반격. 엠버 `allyHit`과 동일 윈도우 공유, 디펜더 수동 운용 충실.
- **freezeZone**: 알레쉬/이본은 냉기 부착 **소모** 동결 / 스노우샤인은 **부착 무관** 동결 → 냉기 부착을 따로 안 깎아 쇄빙·스트라이커 조합에 적합(위키 강제 동결 차별점).
