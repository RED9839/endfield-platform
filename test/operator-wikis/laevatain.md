# 레바테인 (Laevatain / 수르트의 리컨비너) — 열기 / 한손검 / 스트라이커 ★6 (한정)

**최초 6성 한정 열기 스트라이커 · 광역/안정성 특화**. 열기 부착을 **흡수 → 녹아내린 불꽃**(4스택) → **강화 배틀 폭발 + 궁 +100** → 버프형 궁 「황혼」(게이지 **300 = 인게임 최고**). **부활의 불씨**(HP 40%↓ 90% 비호+회복)로 근접 딜러 최고 안정성. 다재다능하나 고점·사이클 경직이 단점. 주스탯 **지능**, 보조 **힘**. 살카즈(로도스, 관리자 직속).

---
## 스킬 (설명문 원문)

### 기본 공격 「재」
최대 5단 열기. 강력한 일격 18 불균형. **처형 400→900%**.

### 배틀 스킬 「불타오르는 화염」 (게이지 100)
> 몰튼 코어 → 전방 지속 열기 + 명중 시 **녹아내린 불꽃 1스택**. **4스택 시 전부 소모 → 광역 추가 공격 + 5초 강제 연소 + 궁 에너지 100**. 궁 중 강화.
- 초기 폭발 62→140% / 지속 6→14% / 추가 342→770% / 궁 중 1·2·추가 147·164·400%(→330·370·900%)

### 연계 스킬 「열화」 (쿨 10~9초)
> 적이 **연소/부식** 상태일 때. 모든 연소/부식 적 발밑 불꽃 열기. 명중당 녹아내린 불꽃 1 + 궁 에너지.
- 240→540% / 명중 1/2/3+명 궁 25/30/35 / 쿨 10→9초

### 궁극기 「황혼」 (게이지 **300**, 쿨 10초)
> 열화의 마검 소환 + 메인 전환. 15초 평타/배틀 강화, 3단 평타 **열기 부착**. (버프형)
- 강화 평타 1·2·3·4단 65·81·115·203% → 146·182·260·456%

## 재능 (설명문 원문)
- **불꽃의 심장**(0정~): 메인 강일/처형/처치 명중 시 주변 **열기 부착 흡수 → 녹아내린 불꽃**(최대 4). 4스택 시 열기 저항 10(0정)→20(3정) 무시(20초).
- **부활의 불씨**(2정): HP 40% 이하 시 **90% 비호 + 매초 최대 HP 5% 회복**, 4(2정)→8초(3정). 120초 1회.
- 장비 조합 I/II/III(파랑/보라/노랑).

## 잠재능력 (명함 제외)
01 녹아내린 마음(강화 배틀 ×1.2·게이지 +20) / 02 기억을 찾는 여정(지능 +20·평타 +15%) / 03 추억의 조각(연소 +50%·×1.5) / 04 아이스크림 용광로(궁 에너지 -15%) / 05 존재의 증명(궁 평타 ×1.2·처치 시 지속 +1초)

## 🎮 현재 구현 (DD류 — `app/game/dd/roster.ts` laevatain, Lv1 계수)
- OP_BASE: cls **striker**, hp 2689, attack 110, speed 64, ultCost **300**(인게임 최고). 주스탯 지능.
- **불꽃의 심장 흡수(엔진 id훅)** — 레바테인의 **일반공격(강일)·배틀·연계 명중 시** 타겟 열기 부착을 흡수 → **녹아내린 불꽃**(`procCount`, 흡수 N + 배틀/연계 자체 명중 1). 일반공격은 흡수만(+0 자체). 검증: ① 일반공격 열기2 흡수→불꽃2 ② 배틀 흡수1+명중1=4 → 강화 폭발 ③ 연계 명중+1 ✅
- **불타오르는 화염**(배틀, power 0.62) — 흡수로 **4스택 도달 시 강화 폭발**(추가 342% + 강제 연소 + 궁 +100 + 불꽃의 심장 열기 증폭). 검증: 강화 폭발·강제 연소·궁 +107 ✅
- **열화**(연계, power 2.40, `target:row`, cooldown 2, `requires` 연소/부식) — 광역 열기 + 흡수 빌드 + id훅 궁충(명중 수 25/30/35). 검증: 게이트 true·궁충 ✅
- **황혼**(궁, power 3.0 즉발 도마, `attach:heat`, ultCost **300**) — **변신**: id훅으로 `timers.twilight` 3턴(≈15초) 부여. 지속 동안 ① **강화 일반공격 ×3 + 광역화**(위키 강화 평타 464%/일반 157%≈2.95 · `pickTargets`가 전체 적 반환) ② **강화 배틀**(1단계 62→147%=+0.85 · 추가 공격 342→400%). 검증: 평타 55→165(×3.0)·광역(적B도 피해)·배틀 궁중 400% ✅
- 엔진 신규: **불꽃의 심장 흡수**(attack/battle/link 공통 — 열기 부착→녹아내린 불꽃, 배틀 4스택 강화 폭발+강제 연소+궁100), **황혼 변신**(`twilight` 타이머 → 평타 ×3 광역·배틀 궁중 배율), **부활의 불씨**(HP 40%↓ 피격 → 90% 비호+회복, `embersCd` 120초 쿨). 검증: 변신 ×3 광역·배틀 강화·비호 0.9 ✅
- ⚠ 단순화: 궁 변신 15초→3턴, 강화 평타 4단/배틀 2단계는 단일 액션 ×배수로 추상화(위키 표 비율 정합), 처치 시 흡수(공격 명중 흡수로 통합), 4스택 열기 저항 무시(amp.heat 근사), 녹아내린 불꽃 만료(소모 전 보유), 잠재.

### 🔑 녹아내린 불꽃 흡수 사이클 (궁 300 충전)
파트너(울프가드 연소·카뮤 열기·아델리아 부식)가 열기 부착/연소 깔면 → 레바테인이 **일반공격/배틀/연계 어느 것으로든 흡수**해 녹아내린 불꽃 누적 → **4스택 배틀 = 강화 폭발 + 궁 +100** → 연계 열화 궁충 → **궁 황혼(300)**. 부착을 직접 안 쌓고 흡수하는 구조라 열기팟 전용. 부활의 불씨는 향후 생존형 오퍼에 재사용.
