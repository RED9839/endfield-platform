# 이본 (Yvonne) — 냉기 / 권총 / 스트라이커 ★6 (한정)

**치명타 기반 말뚝딜 변신 궁 + 강제 동결**. 냉기/자연 부착을 **소모해 강제 동결**(배틀) → 동결 적 강일 시 연계(꽁꽁이 흡인 몹몰이) → **치명타 변신 궁 「아이스 슈터」**(7초 강화 평타·치확/치피 누적, 풀히트 2400%급). **빙점**(냉기/동결 적 치피↑). 간결한 부착-배틀-동결-궁 구조. 쇄빙 파티 동결 공급. 주스탯 **지능**, 보조 **민첩**. 와이번(엔드필드, 4번 협곡·타타 개발자).

---
## 스킬 (설명문 원문)

### 기본 공격 「점프 트리거」
최대 5단 냉기. 강력한 일격 17 불균형. **처형 400→900%**.

### 배틀 스킬 「얼음 폭탄 · β형」 (게이지 100)
> 냉각탄 발사 폭발 냉기. **냉기/자연 부착 적 명중 시 모든 아츠 부착 소모 + 강제 동결 + 소모 스택 비례 냉기**. 동결 부여 후 소모 스택 비례 궁 에너지(다수 명중해도 1회).
- 기초 111→250% / 동결 부여 시 67→150% / 스택당 89→200% / 궁충 동결 10 + 스택당 30

### 연계 스킬 「꽁꽁이 · υ37」 (쿨 20~18초)
> 메인이 **동결 적**에 강력한 일격 후. 꽁꽁이 배치 4충격파 + **흡인(몹몰이)** + 자폭 **강제 동결**. 명중 후 궁 +10(다수여도 1회).
- 충격파 45→100% / 폭발 89→200% / 쿨 20→18초

### 궁극기 「아이스 슈터」 (게이지 **220**)
> 삐삐 배치 + 메인 전환. 7초간 강화 일반공격, 평타마다 **치확 +3%(최대 10스택=+30%)**, 만스택 시 **치피 +60%**. 마지막 평타 = 강일. 동결 적이면 추가 냉기 1회 + **동결 소모**.
- 평타 8.9→20% / 강일 133→300% / 추가 267→600%

## 재능 (설명문 원문)
- **하이테크 버스트**(1정): 배틀이 동결 발동 후 다음 일반공격 즉시 강일(2정 +50%).
- **빙점**(2정): **냉기 부착 적 치피 +10%(2정)→+20%(3정), 동결 적은 ×2**(=+40%).
- 장비 조합 I/II/III(파랑/보라/노랑).

## 🎮 현재 구현 (DD류 — `app/game/dd/roster.ts` yvonne, Lv1 계수)
- OP_BASE: cls **striker**, hp 2689, attack 110, speed 66, ultCost **220**. 주스탯 지능.
- **얼음 폭탄·β형**(배틀, power 1.11, `iceBomb:true`) — 냉기/자연 부착 적이면 **전부 소모 → 강제 동결 + (67%+스택당 89%) 냉기 + 궁충(10+스택×30)**. 검증: 냉기2+자연1=3 소모→동결 3·궁 +100 ✅
- **꽁꽁이·υ37**(연계, power 1.34, `target:row`, cooldown 4, `requires` 동결 적) — 광역 냉기 + 몹몰이 + `apply` **자폭 강제 동결** + id훅 궁 +10. 검증: 게이트 true·동결 유지·궁충 ✅
- **아이스 슈터**(궁, power 6.0+동결 소모 2.67, ultCost 220) — **치명타 변신**(치확 +0.30·치피 +0.60) + **동결 소모 추가 267%**. 검증: 동결 적 궁 1454(변신+빙점+소모) ✅
- 엔진 신규: **`iceBomb`**(냉기/자연 소모 강제 동결+궁충), **빙점**(이본 vs 냉기 적 치피 +0.20/동결 +0.40), **아이스 슈터 변신**(궁 치확 +0.30·치피 +0.60), **동결 소모 추가타**. 빙점 판정은 소모 전 상태(`yvFrozen`/`yvCryo`) 캡처.
- ⚠ 단순화: 궁 7초 말뚝딜(즉발 단일 누킹 + 만스택 치확/치피 즉시 부여로 근사), 하이테크 버스트(동결 후 즉발 강일), 연계 흡인 위치, 잠재.

### 🔑 강제 동결 + 빙점 치명타 (라라와 대비)
배틀 얼음 폭탄이 **냉기/자연 부착을 소모해 강제 동결**(쇄빙 트리거·궁충) → 동결 적에 빙점(치피 +40%) + 변신 궁(치확/치피 만스택)으로 **치명타 폭딜**. 라스트 라이트(냉기 부착 누적 소모)와 달리 **부착을 동결로 전환·소모**하는 구조 → 라라와 동결/부착 경쟁(더블 스트라이커 비효율, 위키). `iceBomb`/빙점은 향후 동결·치명 특화 오퍼에 재사용.
