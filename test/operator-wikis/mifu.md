# 미브 (Mi Fu) — 물리 / 양손검 / 가드 ★6

주스탯 **힘**, 보조 **의지**. 청파 삼형(단운→추형→개천 3스탠스) 강타 딜러 + 물리취약 연계. 자체 방불 부여는 궁뿐 → **팀 방불 보조 필수**(진천우 궁합 최고). (평가상 "스트라이커에 가까운 가드"라 불리나 공식 직군은 가드)

---
## 스킬 (설명문 원문)

### 기본 공격 「검권 합일」
최대 4단 물리. 강일 **25 불균형**(양손검 높음). 처형 400→900%.

### 배틀 스킬 「청파 삼형」 (게이지 100→50→50, 3스탠스)
> 연속 사용 시 3가지 초식. 최초 = **단운**.
- **단운**(게이지 100, **50 반환**): 포승줄 발사, 목표+주변 물리 + **끌어오기(몹몰이)**. → 일정 시간 내 추형 전환. 67→150%
- **추형**(게이지 50): 연속 주먹 물리 + 마지막 **강타**. **강타로 방불 3+ 소모 시** → 개천 전환. 89→200%, 불균형 5
- **개천**(게이지 50): 전방 대량 물리, **강타 피해로 간주**(배틀 피해 아님 → 연타 미적용). 400→600%, 불균형 10

### 연계 스킬 「후회 없는 주먹」 (쿨 20~19초, 조건: 적 방어 불능 3스택+)
> 올려치기 물리 + **물리 취약**(16초). 사용 후 배틀 = 추형 전환.
- 111→250% / 불균형 10 / 물취 5%

### 궁극기 「절심」 (게이지 80)
> 돌진 강제 **띄우기** → 내리꽂아 강제 **넘어뜨리기** + 물리. 사용 후 배틀 = 추형 전환.
- 311→700% / 불균형 20. (딜 기댓값 낮음 — 방불 부여 + 추형 전환 용도)

## 재능 (설명문 원문)
- **냉정**: 개천 명중 시 목표가 **물리취약 또는 불균형**이면 피해 ×1.1(1차)/**×1.2(2차)**.
- **분노**: 연계 후 최대 생명력 **15%(1차)/30%(3차)** 보호막 + 행동 안 끊김(10초, 60초당 1회).

## 잠재능력 (명함 제외)
01 영원한 순례(연계 쿨-2초·물취+5%·+4초) / 03 전투 태세(분노 강화) / 04 삼절의 극의(궁 에너지-15%) / 05 숲을 가르는 주먹(배틀 ×1.1 + 궁 불균형+5)

## 🎮 현재 구현 (DD류 — `app/game/dd/roster.ts` mifu, Lv1 계수)
- OP_BASE: cls **guard**(공식), hp 2689, attack 110(튜닝), speed **46**(민첩 낮음), ultCost 80.
- **청파 삼형 = 3스킬 + 스탠스 시스템**(엔진 신규):
  - **단운**(`gaugeRefund:50`, `setStanceTo:1`) — 몹몰이(row) + 추형 전환.
  - **추형**(`gaugeCost:50`, `requiresStance:1`, `crush`, `stanceFromCrush`) — 강타. **방불 3+ 소모 시 stance 2**(개천 해금).
  - **개천**(`gaugeCost:50`, `requiresStance:2`, `vsWeak:0.2`) — 주력 딜 + **냉정**(물취/불균형 적 ×1.2).
- **후회 없는 주먹**(연계, `requires physBreak>=3`, cooldown 4, `setStanceTo:1`, apply 물취 0.05) — 추형 전환.
- **절심**(궁, `knockdown`, `setStanceTo:1`, ultCost 80) — 방불 부여 + 추형 전환.
- 엔진 신규: `stance`(2턴 윈도우, expire→0) / `gaugeCost`·`gaugeRefund`(가변 게이지) / `vsWeak`(냉정류 조건부 증댐).
- 검증(`tsx test/dd-arts.ts`): 초기 추형/개천 불가 → 단운 후 stance 1(추형 가능) → 추형(방불3 소모) stance 2(개천 가능). ✅
- 미구현: **분노 보호막**(보호막 자체는 있으나 연계 후 자동 부여 미배선), 기본공격 처형, 잠재.
