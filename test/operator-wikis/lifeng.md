# 여풍 (Lifeng) — 물리 / 장병기 / 가드 ★6

주스탯 **민첩**, 보조 **힘**. 물리 파티 올라운더 — 방어 불능 누적 + 물리 취약(딜 보조) + 몹몰이 + 연타 서브딜.

---
## 스킬 (설명문 원문)

### 기본 공격 「업보 파괴」
적에게 최대 4단 물리. 메인 컨트롤 시 강력한 일격 19 불균형.
- 배율(Lv1→12): 1단 24%→55% … 4단 68%→152% / 낙하 80%→180% / 처형 400%→900%

### 배틀 스킬 「신체 정화」 (게이지 100)
> 전방으로 창을 여러 번 휘둘러 2회의 물리 피해. 이후 힘차게 지면을 내리쳐 전방 모든 적에게 다시 물리 피해 + **넘어뜨리기**. **마지막 공격에 명중한 적이 방어 불능 상태가 아닐 경우** 추가로 12초간 **물리 취약**.
- 배율 1·2단계 38%→86%(각), 3단계 119%→268% / 3단계 불균형 10 / 물리 취약 5%→12%
- **AoE(전방)**. ⚠ 물리 취약은 **방어 불능 스택 없을 때만** 발동(막타 빗나가면 미부여). 잠재1 파집: 물취 +5% + 방불 2스택 이하로 조건 완화

### 연계 스킬 「분노의 형상」 (쿨 16~15초, 조건: 물리 취약/갑옷 파괴 적이 메인 강일 받음)
> 화신을 내세워 장창으로 찌르며 물리 피해 + **20초 연타 획득**.
- 배율 1단계 47%→105%, 2단계 167%→375% / 불균형 10. **넘어뜨리기 없음 — 물리 + 연타만**

### 궁극기 「움직이지 않는 마음」 (게이지 90)
> 부동저로 지면 내리침. 넓은 범위 모든 적 물리 + **넘어뜨리기** + **몹몰이(끌어당김)**. 일정 시간 후 범위 내 남은 적 다시 물리 + 넘어뜨리기. **연타 소모 시 대량 추가 물리.**
- 1·2단계 178%→400%(각), 추가 267%→600% / 불균형 5/5/5

## 재능 (설명문 원문)
- **돈오**: 지능 및 의지 1포인트당 추가 공격력 **+0.15%**(주/보조 스탯 보너스에 합산 = 장비 등급 비례 화력).
- **복마**: 적에게 **넘어뜨리기** 피해를 줄 때마다 추가로 자신의 공격력 **100%** 물리 피해.

## 잠재능력
- 01 파집: 신체 정화 물리 취약 +5%, 방불 2스택 이하 적에게도 발동(조건 완화). **1돌 권장**(명함↔1잠 차이 최대급).
- 05 불해: 복마 강화 — 15초마다 다음 발동 시 공격력 250% 추가 물리 + 불균형 5.

## 🎮 현재 구현 (DD류 — `app/game/dd/roster.ts` lifeng, Lv1 계수)
- **신체 정화**(배틀, power 1.95=38+38+119%, `knockdown`, staggerVal 10, `selfPhysBonus 1.0`) — AoE 넘어뜨리기 + **복마** + 물리취약 0.05(`apply`: `physBreak===0`일 때만 = 위키 조건). 넘어뜨리기는 배틀/궁만(연계는 anomaly 없음 → 자동 제외).
- **분노의 형상**(연계, power 2.14=47+167%, cooldown 3턴=16초) — `requires: vulnFor(physical)>0 || armor-break`(물리취약/갑옷파괴 적). `apply`: multiHit+1(연타). 넘어뜨리기 없음.
- **움직이지 않는 마음**(궁, power 3.56=178+178%, staggerVal 15, `knockdown`, `selfPhysBonus 1.0`, selfUlt) — 광역 넘어뜨리기 몹몰이 + 복마 + 연타 소모(엔진 MH_ULT). ultCost 90.
- 엔진: **복마** = `applyAnomaly`에서 `selfPhysBonus` → 넘어뜨리기마다 `attack×eb×1.0` 추가(위키 100% 그대로, 카드판 0.6 보정과 달리 정합). **돈오** = attack 스탯에 baked(지능+의지 파생).
- 검증(`tsx test/dd.ts`): 신체정화 넘어뜨리기마다 복마 발동, 방불 0 적에만 물리취약, 보스 9R 승리. ✅
- 미구현: 기본공격 처형, 잠재(파집 1돌 등 — 명함 baseline 제외).
