# 여풍 (Lifeng) — 물리 / 가드 ★?

※ 위키 원문 배율표는 미보유(구현 기반 복원). 넘어뜨리기 + 물리 취약 + 스탯 파생 화력. physBreak **build/knockdown**.

## 스킬
- **배틀 신체 정화** (AoE): 창 2회 물리 → 지면 내리침 전방 물리 + **넘어뜨리기**. 마지막 명중 적이 방어 불능 아니면 물리 취약 부여.
- **연계 분노의 형상** (cond: 물리 취약/갑옷 파괴 적이 강일 받음): 화신 장창 찌르기 물리 + 연타.
- **궁극 움직이지 않는 마음** (AoE): 부동저 광역 물리 + 넘어뜨리기 + 몹몰이(끌어당김).

## 재능
- **돈오**: 지능·의지 → 공격력 변환(스탯 파생 화력 = 장비 등급 비례).
- **복마**: 넘어뜨리기 발동 시 공격력 100% 추가 물리.
- (신체 정화 물리 취약 + 분노의 형상 연타)

## 🎮 현재 구현
- passive: `{ statPower: 0.12, knockdownBonus: 0.6, appliesPhysVuln: 0.12, grantMultiHit: true }`
  - `statPower 0.12` = 돈오(자기 화력 +지능·의지=장비 등급 비례)
  - `knockdownBonus 0.6` = 복마(넘어뜨리기 시 공격력 ×0.6 추가 물리)
  - `appliesPhysVuln 0.12` = 신체 정화(방어 불능 없는 적에 물리 취약, 배틀)
  - `grantMultiHit` = 분노의 형상 연타
- physBreak build/knockdown. ULT_ENERGY 90
