# 명일방주: 엔드필드 — 전투 시스템 (밸패/구현 레퍼런스)

전투 시스템 wiki 정합 기준. DD류 리뉴얼(`app/game/dd/`)의 메커니즘 출처.

---
## 불균형치 (스태거)
적 체력 밑 하얀 게이지. 강력한 일격/배틀/연계/궁/패턴 끊기로 누적(패턴 끊기가 최대).
- **가득 차면 불균형 상태**: 적 행동 불가 + **받는 피해 +30%** + 근처 일반공격으로 **처형** 가능.
- 일부 정예/보스는 중간에 **불균형 지점**(넘으면 잠시 행동 중단).
- 차지(보스 붉은 원): 배틀/연계/궁으로 끊으면 대량 불균형치 + 경직.

## 물리 피해
### 방어 불능 (최대 4스택, 20초 미갱신 시 소멸)
- **자체 효과 없음.** 강타/갑옷파괴로 소모하거나 띄우기/넘어뜨리기 효과 발동.
- 방불 스택과 아츠 부착 스택은 **따로 계산**.

### 물리 이상 (띄우기·넘어뜨리기·강타·갑옷파괴)
**핵심 규칙: 방불 0인 적엔 이상 효과 미발동 → 방불 1스택만 부여. 방불 있으면 효과 발동.** (강제 부여 시 스택 무관)
- 물리 이상 계수 = 발동 오퍼 스탯(레벨 0.25%/lv, 아츠 강도 1%/pt).
- **띄우기/넘어뜨리기**: 방불 1스택 부여. 방불 상태면 **120% 물리 + 불균형 10** + 약한 적 공중에 뜸/넘어짐.
- **강타**: 방불 0이면 1스택 부여. 방불 상태면 **전부 소모 → 300/450/600/750%** (1/2/3/4스택).
- **갑옷 파괴**: 방불 0이면 1스택. 방불 상태면 **전부 소모 → 100/150/200/250% + 받는 물리 +12/16/20/24%** (12~30초).

## 아츠 피해 (열기·전기·냉기·자연) — ⚠ DD 미구현(후속)
### 아츠 부착 (속성별 최대 4스택, 20초)
자체 효과 없음.
- **아츠 폭발**: 같은 속성 2스택 이상 → 160% 피해(부착 미소모).
- **아츠 이상**: 서로 다른 2종 부착 → 모든 부착 소모 발동(나중 부착 속성으로 결정).
  - **연소**(+열기): 160~400% 열기 + 10초 지속피해.
  - **감전**(+전기): 160~400% 전기 + 받는 아츠 +12~24%.
  - **동결**(+냉기): 130% 냉기 + 약한 적 6~9초 빙결.
    - **쇄빙**: 동결 + (방불/물리이상) → 동결 소모 **240/360/480/600% 물리**(아츠 이상이나 물리 피해).
  - **부식**(+자연): 160~400% 자연 + **모든 속성 저항**(물리 포함) 감소.

## 자원
- **스킬 게이지**: 파티 공유. 배틀 100 소모. 12.5초/칸 자연 회복 + 강력한 일격/처형/극한회피로 회복. 일부 오퍼 반환.
- **궁극 에너지**: 배틀 사용 시 (소모 게이지/100)×6.5, 연계 시 시전자 10. 반환분 사용 시 충전 감소.

## 전투 효과/용어
- **증폭**(주는 피해↑)/**취약**(받는 피해↑): 중첩 가능, 상위가 하위 포함(물리취약 < 취약).
- **연타**: 다음 **배틀** 피해 +30/45/60/75% 또는 **궁** +20/30/40/50%. ⚠ **"배틀/궁 피해"에만** — 관리자 강타·강화 일반공격엔 미적용.
- **허약**(공격력↓, 곱연산)/**비호**(받는 피해↓, 최고값만)/**보호**(보호막).
- **오리지늄 결정**(관리자 전용): 약한 적 봉인 + 결정 부착 판정. 물리 이상/궁으로 소모 시 추가 물리.
- **이상 레벨 +1**: X스택 소모 시 X+1로 판정(최대 4).
- **소모/반환**: 강타=방불 소모, 연소=부착 소모 / 반환은 "획득"으로 미판정.

## 자원 경제 상수 (`combat.ts`)
GAUGE_COST 100(배틀) · GAUGE_REGEN 45/라운드 · BASIC_RECOVER 35(평타) · EXEC_RECOVER 60(처형) · EXECUTE_MULT 6 · ULT_BATTLE 6.5(전체) · ULT_LINK 10(시전자). 시작 게이지 200/최대 300.

## 지속시간 (턴, **1턴 ≈ 5초**) — 재적용 시 리셋(갱신형)
`u.timers: Record<key, 잔여턴>`, `startRound`에서 감쇠 → 0이면 `expire`. 위키 초 ÷ 5 환산.
| 효과 | 위키 | 턴 |
|---|---|---|
| 방어 불능 / 아츠 부착 | 20초 | **4** |
| 본질붕괴(atkBuff) / 취약 / 부식 / 일반 버프 | 15·12~24초 | **3** |
| 연소 DoT / 동결 | 10·6~9초 | **2** |
| **연계 쿨타임**(`linkCd`, `cooldown ?? 3`) | 15~16초 | **3** |

- **갱신**: 콤보로 계속 때리면 타이머 리셋(위키 "미갱신 시 소멸").
- **정화**: 디버프 타이머만 제거(버프 유지).
- **연계**: 조건(usage gate) + 쿨타임 **둘 다** 충족해야 사용.
- 검증: `tsx test/dd-duration.ts` — 12/12(방불 4턴 소멸·갱신·연계 쿨 3턴·본질붕괴/취약/연소 만료·정화).

## 직군 + 방어 스탯 레이어
| 위키 | 구현 | 상태 |
|---|---|---|
| **직군 6종**(가드/캐스터/스트라이커/뱅가드/디펜더/서포터) | `cls: DDClass` | ✅ 필드 |
| **방어력 → 받는 피해 ×(100/(def+100))** (140=58.3%↓) | `mitigate` `u.defense` | ✅ |
| **물리/아츠 저항**(민첩→물리·지능→아츠, 1=1%↓) | `u.resist.{physical,arts}` `mitigate` | ✅ |
| 부식(전 저항 감소) ↔ 저항 | vuln.all로 상쇄 | ✅ 토대 마련 |

기본값 0 → 밸런스 무변(장비·디펜더 이식 시 값 부여). 직군 매핑: 가드=물리이상, 캐스터=`applyAttach`, 스트라이커=관리자류, 뱅가드=게이지 회복, 디펜더=`shield`/`cleanse`, 서포터=`amp`/`vuln`/`weaken`.

미구현(보류): 주/보조 능력치→공격력 파생, 6스탯 시트, 치명타(결정론이라 기댓값 예정), 아츠강도(attack로 근사).
