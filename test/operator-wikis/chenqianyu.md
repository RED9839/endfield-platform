# 진천우 (Chenqianyu) — 물리 / 한손검 / 가드 ★5

주스탯 **민첩**, 보조 **힘**. 물리/쇄빙 파티 **메인 누커** — 띄우기 방어 불능 누적 + 차지 캔슬 최강 + 초고계수 단일 궁(보스 삭제기). 물리덱 사실상 필수.

---
## 스킬 (설명문 원문)

### 기본 공격 「파비하」
적에게 최대 5단 물리. 메인 컨트롤 시 강력한 일격 16 불균형. (평타 공속 매우 빠름 → 강일 트리거 캐릭과 궁합 최고)
- 배율(Lv1→12): 1단 20%→45% … 5단 40%→90% / 낙하 80%→180% / 처형 400%→900%

### 배틀 스킬 「귀궁우」 (게이지 100)
> 목표한 적을 올려 칩니다. 물리 피해 + **띄우기**.
- 배율 169%→380% / 불균형 10 / 에어본 1~2.5초. **단일**

### 연계 스킬 「견천하」 (쿨 16~15초, 조건: 적이 방어 불능 상태)
> 적을 관통하고 돌진 베기. 경로 상의 모든 적에게 물리 피해 + **띄우기**.
- 배율 120%→270% / 경로 **AoE**. **게이지 미소모**(연계) → 방불 빠른 누적 + 게이지 유연성

### 궁극기 「예풍상」 (게이지 70)
> 목표한 적에게 7단 베기. 각 물리 + 마지막 베기 대량 물리.
- 베기 36%→81%(×6) + 마지막 455%→1023% = **총 1509%(마스터리)** / 불균형 15+20. **순수 단일 누킹(보스 삭제기, 띄우기 없음)**. 게이지 70으로 매우 저렴

---
## 재능 (설명문 원문)
- **칼날 베기**: 스킬이 적에게 명중할 때마다 공격력 **+8%**, 10초 지속, 최대 5스택.
- **흐름 끊기**: 스킬로 적의 차지를 끊었을 때 추가로 **불균형 +10**.

## 잠재능력
- 01 절영: HP 50% 이하 적 피해 +20% / 02 가문의 무술: 민첩+15, 물리 피해+8% / 03 쌍검의 협객: 배틀·연계·궁극 배율 ×1.1 / 05: 견천하 쿨 -3초

## 🎮 현재 구현 (DD류 — `app/game/dd/roster.ts` chenqianyu, Lv1 계수)
- **귀궁우**(배틀, power 1.69=169%, `launch`, staggerVal 10) — 올려치기 띄우기, 자체 방불 빌드.
- **견천하**(연계, power 1.2=120%, `launch`, `requires physBreak>0`, cooldown 3턴=16초, target row=관통) — 방어 불능 적 한정, 게이지 무소모.
- **예풍상**(궁, power 6.71=36×6+455=671%(Lv1), staggerVal 35=15+20, single, selfUlt, ultCost **70**) — 7단 단일 누킹 보스 삭제기. 띄우기 없음.
- 엔진: **칼날 베기** = `rampAtk 0.08`(OP_BASE) → 스킬마다 atkBuff +0.08, 최대 5스택(+40%), 2턴. 본질붕괴와 같은 atkBuff 필드(유닛별 독립).
- 미구현: **흐름 끊기**(차지 끊기 추가 불균형 — 차지 메커니즘 미모델), 기본공격 처형, 잠재(쌍검의 협객 ×1.1 등 — 명함 baseline 제외).
- 검증(`tsx test/dd.ts`): 귀궁우 빌드→견천하(방불 적)→예풍상 보스 삭제(841 단일 막타). 보스 6R 승리. ✅
