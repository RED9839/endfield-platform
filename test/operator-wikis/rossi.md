# 로시 (Rossi) — 물리+열기 / 한손검 / 가드 ★6 (한정)

주스탯 **민첩**, 보조 **지능**. 물리/열기 **하이브리드 딜러** — 띄우기 방불 + 열기 부착 + 늑대의 발톱(DoT·취약) + 치명타 기반 자버프/회복. 방불+아츠부착 **둘 다 필요** → 물리+아츠 혼합 파티(질베르타 최적 서포터).

---
## 스킬 (설명문 원문)

### 일반 공격 「끓어오르는 늑대의 피」
최대 5단 물리. 강일 18 불균형. 처형 400→900%.

### 배틀 스킬 「붉은색의 그림자」 (게이지 100)
> 목표 돌진 물리 + **띄우기**. 목표가 **이미 방어 불능 상태면** 추가로 **울프팀의 진주** 발동 → 돌진 **열기 피해**.
- 1단계 85%→192%(불균형 5) / 2단계(진주, 열기) 128%→288%(불균형 10)

### 연계 스킬 「그림자가 타오르는 순간」 (쿨 15~14초, 조건: 적이 **방어 불능 + 아츠 부착** 동시)
> 연속 2회. 1타 물리. 2타는 **아츠 부착 모두 소모** → 소모 스택 비례 물리 + **띄우기** + 자신 **치명타 확률/피해 증가**. 정확히 연계되면 추가 방불 1스택.
- 1단계 67→150% / 2단계 133→300% / 소모 스택당 +80→180% / 치확 +15→25% · 치피 +30→50%(15초)

### 궁극기 「기습 '날카로운 발톱'」 (게이지 110)
> 다단 열기 찌르기 + 단검 2단 베기 대량 열기 + **열기 부착**. 치명타 시 더 강한 치명타 피해.
- 찌르기 총 275→600% / 1단 베기 111→250% / 2단 베기 333→750% = **총 719%(Lv1)→1600%(Lv12)** / 불균형 25 / 치피 +60%. **단일**

## 재능 (설명문 원문)
- **절흔**: 진주가 피해 후 목표에 **늑대의 발톱**(25초, 중첩X) → 매초 공격력 **30%** 물리 DoT + 받는 물리·열기 피해 **+12%**.
- **끓어오르는 피**: 늑대의 발톱 적에게 스킬 치명타 시 추가 열기(공격력 24%) + [지능×0.08] 회복. 목표가 연소면 ×1.5.

## 잠재능력 (명함 제외)
01 동경의 서막(배틀·연계 ×1.15 + 진주 게이지 +10) / 02 완벽한 간격(민첩+20, 치확+7%) / 04 순수한 소망(궁 에너지 -15%) / 05 전설의 종착지(궁 ×1.1 + 치피+30%)

## 🎮 현재 구현 (DD류 — `app/game/dd/roster.ts` rossi, Lv1 계수)
- OP_BASE: cls guard, hp 2689, attack 110(튜닝), speed **90**(민첩 최상위), ultCost **110**.
- **붉은색의 그림자**(배틀, power 0.85, `launch`, staggerVal 5) — 돌진 띄우기. `apply`: **이미 방불**(post-띄우기 physBreak>1 ⟺ pre≥1)이면 **늑대의 발톱**(dot=attack×0.3, 5턴 + 물리/열기 취약 0.12). 검증: 1타(방불0) 미발동 / 2타(방불) DoT 33+취약 ✅
- **그림자가 타오르는 순간**(연계, power 2.5, `launch`, cooldown 3, `requires physBreak>0 && 아츠부착`) — 아츠 소모(`arts=0`) + **치명 버프(치확 0.30·치피 1.0, 3턴)**. 검증: 방불만→불가 / 방불+열기부착→가능 ✅
- **기습 날카로운 발톱**(궁, power 7.19=275+111+333%, `element:heat`, `attach:heat`, selfUlt, ultCost 110) — 다단 열기 단일 누킹 + 열기 부착(연소 연계 가능).
- **치명타 정식 구현**(기본 치확 0.05·치피 0.50, 기댓값 ×(1+치확×치피)) → 연계 버프 시 ×1.30. 장비 연동 시 함께 튜닝.
- ⚠ **미모델(근사)**: 궁 "치명 시 추가 치피"·**끓어오르는 피(치명 회복)** — 치명 *조건부* 트리거 + 힐 시스템 의존. 회복 재능 드롭(힐 추가 시 재정합).
- **하이브리드 파티 필요**: 연계가 방불+아츠부착 이중 조건 → 물리 빌더 + 아츠 부착러 동반 필수(위키 정합).
